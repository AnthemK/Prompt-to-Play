# Launcher Icon Assets

This folder stores launcher icon assets for `lite-trpg-sim`.

## Available Icons

- `lite-trpg-sim-launcher-icon.svg`
  - Base version
  - Dark tabletop fantasy with d20 crest + blade + candle glow

- `lite-trpg-sim-launcher-icon-variant-a.svg`
  - Variant A (Minimal Emblem)
  - Cleaner silhouette, stronger icon readability at small sizes

- `lite-trpg-sim-launcher-icon-variant-b.svg`
  - Variant B (Compass Adventure)
  - Medieval oak-table style, parchment map routes, stronger compass rose, tabletop dice accent

## Suggested Usage

- If you want maximum clarity in Finder list/icon views: choose Variant A.
- If you want stronger "adventure" atmosphere: choose Variant B.
- If you want the current balanced style: keep the base version.

## Apply Icon To `launch_game.command` (macOS)

1. Open any SVG icon above with Preview.
2. Export it as PNG (recommended `1024x1024`).
3. In Finder, select the exported PNG and press `Cmd + I`.
4. Click the small icon thumbnail in the top-left of the info window, then press `Cmd + C`.
5. Select `launch_game.command`, press `Cmd + I`.
6. Click its top-left icon thumbnail and press `Cmd + V`.

After that, `launch_game.command` will display the custom icon in Finder.
