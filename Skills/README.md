# Skills（多 Agent 经验共享区）

这个目录用于沉淀“可复用的创作经验”，服务 `prompt-to-play` 下的多项目与多 agent 协作。

## 使用目标

- 让不同 agent 可以快速接手同一项目
- 让已踩过的坑不再重复踩
- 让架构、实现、内容创作经验可复用到新项目

## 当前文件

- `CREATIVE_PLAYBOOK.md`
  - 架构设计、项目管理、代码实现、文本/图片/音频创作经验
- `LITE_TRPG_SIM_DOC_GOVERNANCE.md`
  - 面向多读者项目的文档边界治理经验
- `STAGE_UPDATES.md`
  - 按“阶段”记录新增经验（不是每天流水账）

## 维护规则（建议统一遵守）

- 每完成一个阶段性任务后，更新一次 `STAGE_UPDATES.md`
- 若经验已经稳定，合并进 `CREATIVE_PLAYBOOK.md` 的长期规则
- 每条经验尽量写清：
  - 适用场景
  - 推荐做法
  - 常见坑
  - 最小可执行检查点

## 命名与格式约定

- 文件名使用大写蛇形或短横线，保持可读性
- 用 Markdown，优先中文；需要时可附英文关键词
- 避免空话，优先“可执行原则”
