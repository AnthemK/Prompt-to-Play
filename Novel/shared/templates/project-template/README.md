# Project Template

这是新小说项目的起始模板。

## 推荐复制后保留的核心文件

- `README.md`
- `MEMORY.md`
- `TODO.md`
- `docs/`
- `01-planning/`
- `02-outlines/`
- `03-drafts/`
- `04-revision/`
- `05-release/`

## 推荐使用顺序

1. 先写 `01-planning/project-brief.md`
2. 再写 `docs/world-bible.md` 与 `docs/character-bible.md`
3. 再写 `docs/plot-blueprint.md`
4. 然后进入 `02-outlines/`
5. 章纲稳定后再写 `03-drafts/`
