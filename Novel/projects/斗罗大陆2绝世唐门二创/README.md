# 轻量防出戏设定库

这套库的目标只有一个：在写《斗罗大陆II绝世唐门》同人时，尽量避免明显出戏。

它服务的场景是：
- 写前快速检索
- 列大纲前校验
- 写正文时随手回查
- 写完后做防出戏复查

它不追求的东西是：
- 全量百科
- 微小细节全收录
- 纯资料堆砌
- 脱离写作场景的总结型整理

## 各文件用途

- `00原文材料+参考资料/`：原文、百科缓存与提取文件，只作来源与辅助检索
- `01基础设定/世界与修炼速查.md`：世界规则、修炼规则、斗二时代变化的防错底稿
- `01基础设定/势力学院速查.md`：学院线、唐门线、帝国线、魂导器线的防错底稿
- `02核心角色/核心角色速查.md`：最容易写崩的角色速查
- `03写作约束/不可出戏清单.md`：高频翻车点总表
- `03写作约束/可自由改写部分.md`：哪些地方能大胆改，哪些边界别碰
- `04写作辅助/写作前快速检索模板.md`：开写前 3 分钟自查
- `04写作辅助/章节大纲校验提示词.md`：拿去审大纲，提前发现结构性出戏
- `04写作辅助/正文校验提示词.md`：拿去审正文，找表达层面的违和和冲突

## 推荐使用流程

### 写前看什么

- 第一步：先看 `01基础设定/世界与修炼速查.md`
- 第二步：再看 `01基础设定/势力学院速查.md`
- 第三步：如果这章人物互动很多，再补看 `02核心角色/核心角色速查.md`
- 如果本章重点是人物互动或感情推进，请额外查看 `03写作约束/高频角色互动防错卡.md`

### 列大纲前用什么

- 先填 `04写作辅助/写作前快速检索模板.md`
- 再用 `04写作辅助/章节大纲校验提示词.md` 审一次大纲
- 如果大纲碰到高风险改动，再回看 `03写作约束/不可出戏清单.md`

### 写完正文后用什么

- 先用 `04写作辅助/正文校验提示词.md` 做正文校验
- 再用 `03写作约束/不可出戏清单.md` 复查高频翻车点
- 如果改动较大，再回看 `03写作约束/可自由改写部分.md`，确认没有越界

## 最适合这套库处理的写作场景

- 前期学院成长线
- 唐门复兴线
- 以人物互动为主、但仍在主线时代背景里的章节
- 需要快速判断“这一章会不会明显出戏”的大纲和正文

## 这套库不擅长处理的内容

- 追求微小细节百分百考据的百科式整理
- 需要大量魂导器型号、制度细节、完整时间轴的重度设定文
- 完全脱离《绝世唐门》主气质的高度架空改写
