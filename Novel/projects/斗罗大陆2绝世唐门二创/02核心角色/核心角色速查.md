# 核心角色速查

## 1. 前期写作绝对高频角色

### 霍雨浩
- 基础身份：斗二主角；白虎公爵之子；早年随母姓霍
- 所属势力：唐门、史莱克学院
- 武魂 / 能力基础：灵眸；本体武魂；精神属性；后走双生武魂路线
- 性格底色：隐忍、能吃苦、执念深、重情
- 关系定位：唐门与史莱克主线核心；与王冬 / 王冬儿 / 唐舞桐主感情线绑定
- 不能乱改的点：不能写成天生狂傲型龙傲天；不能丢掉苦出身和精神系成长底色
- 写作提醒：写他时，强大可以增长，但“克制、韧性、情感驱动”不能丢

### 王冬
- 基础身份：霍雨浩前期最重要同龄伙伴
- 所属势力：史莱克学院
- 武魂 / 能力基础：顶级天赋路线人物
- 性格底色：骄傲、讲究、带少年意气
- 关系定位：和霍雨浩关系极近；后续与王冬儿、唐舞桐主线相连
- 不能乱改的点：不能写成普通男同学；不能和霍雨浩关系疏淡
- 写作提醒：前期王冬要有“亮眼、别扭、亲近霍雨浩”的感觉

### 唐雅
- 基础身份：斗二开篇唐门门主
- 所属势力：唐门、史莱克学院
- 武魂 / 能力基础：唐门当代门主与魂师
- 性格底色：活泼、直接、情绪外露
- 关系定位：贝贝恋人；霍雨浩老师；唐门复兴线火种人物
- 不能乱改的点：不能只写成轻松搞笑角色；不能丢掉对唐门的执念
- 写作提醒：她表面跳脱，内里是“唐门不能断在我这里”

### 贝贝
- 基础身份：唐门大师兄；霍雨浩前期引路人
- 所属势力：唐门、史莱克学院
- 武魂 / 能力基础：蓝电霸王龙
- 性格底色：温和、稳重、可靠
- 关系定位：唐雅恋人；霍雨浩大师兄；唐门线稳定器
- 不能乱改的点：不能写成轻浮油滑型
- 写作提醒：贝贝常负责“解释规则、稳住场面、托住后辈”

### 穆恩
- 基础身份：史莱克学院核心宿老
- 所属势力：史莱克学院、海神阁
- 武魂 / 能力基础：顶级强者
- 性格底色：深沉、苍老、眼光极强
- 关系定位：霍雨浩极重要长辈；学院顶层意志代表
- 不能乱改的点：不能写成普通老师
- 写作提醒：穆恩一出场，就要有“学院最高层重量”

### 玄子（玄老）
- 基础身份：史莱克学院核心人物；后期海神阁主
- 所属势力：史莱克学院、海神阁
- 武魂 / 能力基础：顶级强者
- 性格底色：表面随性，实则护短且有责任
- 关系定位：学院顶层；关键时刻代表史莱克立场
- 不能乱改的点：不能只写成爱吃喝的怪老头
- 写作提醒：玄老的随性不能盖过他的担当

## 2. 中期常见关键角色

### 王冬儿
- 基础身份：中期核心女主形态
- 所属势力：史莱克学院一侧
- 武魂 / 能力基础：顶级天赋路线
- 性格底色：较前期更柔和，但感情仍极重
- 关系定位：与霍雨浩主感情线核心绑定；与王冬、唐舞桐主线连续
- 不能乱改的点：不能写成和王冬、唐舞桐完全割裂的新角色
- 写作提醒：若处于王冬儿阶段，要保留前后连续性

### 唐舞桐
- 基础身份：核心女主；唐三、小舞之女
- 所属势力：神界唐家血脉；人间线深连史莱克与唐门
- 武魂 / 能力基础：顶级天赋路线
- 性格底色：明艳、骄傲、感情浓烈
- 关系定位：霍雨浩官配核心；王冬 / 王冬儿最终归并点
- 不能乱改的点：不能和霍雨浩拆成普通暧昧对象；不能脱离唐三之女身份
- 写作提醒：既要有少女感，也要有高位出身带来的强势

### 徐三石
- 基础身份：史莱克七怪成员
- 所属势力：史莱克学院、唐门
- 武魂 / 能力基础：防御路线强者
- 性格底色：嘴上不正经，实际扛事
- 关系定位：团队气氛位与关键补位
- 不能乱改的点：不能只写成谐星
- 写作提醒：他适合写反差感，不适合写成纯搞笑工具人

### 江楠楠
- 基础身份：史莱克七怪成员
- 所属势力：史莱克学院、唐门
- 武魂 / 能力基础：敏攻路线
- 性格底色：清冷、自持、克制
- 关系定位：徐三石感情线核心；团队稳定成员
- 不能乱改的点：不能写成轻浮热闹型
- 写作提醒：她的情绪表达通常收着来

### 和菜头
- 基础身份：史莱克七怪成员；唐门魂导体系关键人物
- 所属势力：史莱克学院、唐门
- 武魂 / 能力基础：魂导师路线核心人物
- 性格底色：厚道、可靠、执行力强
- 关系定位：唐门现代化、魂导器线的重要支点
- 不能乱改的点：不能忽略他在魂导器线的重要性
- 写作提醒：涉及唐门魂导堂、魂导器推进时，他常是关键人物

### 萧萧
- 基础身份：史莱克七怪成员
- 所属势力：史莱克学院、唐门
- 武魂 / 能力基础：控制、辅助意义都强
- 性格底色：细腻、带明显少女感
- 关系定位：早期小队核心成员之一
- 不能乱改的点：不能边缘化成可有可无
- 写作提醒：她适合承担团队内部情绪连接

### 马小桃
- 基础身份：史莱克内院代表人物
- 所属势力：史莱克学院
- 武魂 / 能力基础：火属性顶级路线
- 性格底色：炽烈、强势、压场
- 关系定位：学院强者线、内院线高识别度人物
- 不能乱改的点：不能写成温吞型
- 写作提醒：她的存在感来自“强烈”和“爆发”

### 言少哲
- 基础身份：史莱克学院高层；武魂系核心人物
- 所属势力：史莱克学院
- 武魂 / 能力基础：顶级强者
- 性格底色：强者气场明显，学院立场鲜明
- 关系定位：学院管理层与高层决策位
- 不能乱改的点：不能写成普通导师
- 写作提醒：涉及学院高层决断时，他是高频出场位

### 橘子
- 基础身份：日月帝国战争线关键女性角色
- 所属势力：日月帝国
- 武魂 / 能力基础：更偏军略、执行与现实选择
- 性格底色：冷静、能忍、复杂
- 关系定位：与霍雨浩有重大纠葛；与日月帝国权力线深绑
- 不能乱改的点：不能写成单纯恶毒女配
- 写作提醒：写橘子时，复杂性比“坏”更重要

### 徐天然
- 基础身份：日月帝国核心统治者
- 所属势力：日月帝国
- 武魂 / 能力基础：帝王位格人物
- 性格底色：阴沉、狠、权欲重
- 关系定位：日月帝国扩张与压迫感的集中体现
- 不能乱改的点：不能写成热血皇帝型角色
- 写作提醒：写日月威胁时，徐天然常是压迫源头

## 3. 仅在涉及特定主线时才需要重点关注的角色

### 镜红尘 / 孔德明
- 基础身份：日月帝国魂导体系高层代表
- 所属势力：日月帝国
- 武魂 / 能力基础：魂导器体系顶层人物
- 性格底色：老辣、务实、重现实力量
- 关系定位：日月帝国技术与制度压迫感的重要来源
- 不能乱改的点：不能写成只会叫嚣的空壳反派
- 写作提醒：涉及日月高层魂导器线时再重点回查

### 龙逍遥
- 基础身份：龙皇斗罗；圣灵教一侧顶级强者
- 所属势力：与圣灵教阵营纠缠很深
- 武魂 / 能力基础：极限层面强者
- 性格底色：沉重、复杂
- 关系定位：圣灵教高端战力与旧时代宿命线人物
- 不能乱改的点：不能写成纯嗜杀模板反派
- 写作提醒：他适合写复杂立场，不适合写平面恶人

### 叶夕水
- 基础身份：死神斗罗；圣灵教一侧最可怕人物之一
- 所属势力：圣灵教
- 武魂 / 能力基础：极限层面强者；双生武魂路线
- 性格底色：阴冷、强势、极端
- 关系定位：圣灵教顶层黑暗源头之一
- 不能乱改的点：不能削成普通邪教头目
- 写作提醒：她一出场，应有极强压迫感和黑暗感

### 钟离乌
- 基础身份：圣灵教核心人物
- 所属势力：圣灵教
- 武魂 / 能力基础：邪魂师顶层战力
- 性格底色：阴狠、冷酷
- 关系定位：圣灵教组织性与进攻性代表
- 不能乱改的点：不能写成无能狂怒型反派
- 写作提醒：需要写圣灵教威胁时，再重点查他

### 毒不死
- 基础身份：本体宗宗主
- 所属势力：本体宗
- 武魂 / 能力基础：本体宗顶层强者
- 性格底色：强硬、霸道、有骨气
- 关系定位：本体武魂与本体宗路线核心人物
- 不能乱改的点：不能弱化本体宗在本体武魂领域的分量
- 写作提醒：只在本体宗或本体武魂主线里重点回查

### 帝天
- 基础身份：兽神；星斗大森林统治级魂兽
- 所属势力：星斗大森林魂兽阵营
- 武魂 / 能力基础：当世第一魂兽级别存在
- 性格底色：威严、冷酷、背负守护责任
- 关系定位：魂兽阵营最高压迫感之一
- 不能乱改的点：不能写成普通十万年魂兽头目
- 写作提醒：涉及星斗核心线时再重点查他

### 天梦冰蚕
- 基础身份：霍雨浩最早期最大外挂与陪伴者；智慧魂环
- 所属势力：与霍雨浩绑定
- 武魂 / 能力基础：精神与冰双属性；精神层面极强
- 性格底色：话多、自恋、搞怪，但真心帮霍雨浩
- 关系定位：霍雨浩成长线关键伴随者
- 不能乱改的点：不能只写成搞笑宠物
- 写作提醒：写他时要同时保留“吐槽感”和“关键设定承载”

### 唐三
- 基础身份：唐门先祖；神界执法者；唐舞桐之父
- 所属势力：神界、唐门源头
- 武魂 / 能力基础：神级存在
- 性格底色：强势、深谋、护短
- 关系定位：唐舞桐父亲；霍雨浩的考验者与高位观察者
- 不能乱改的点：不能写成纯反派，也不能写成毫无布局的慈父
- 写作提醒：唐三出场时，重点是“高位、控制力、父亲立场”
