# 05 Drafts

这个目录只处理正文草稿。

## 这个目录解决什么问题

如果你是新协作者，这一层主要回答：

- 正式正文现在写到哪里了
- 已经落文的章节长什么样
- 你若要做对照实验，应拿哪份正式文本做参照

它不处理：

- 卷纲
- 章纲
- 逐章功能表
- 正文细纲
- 伏笔账本

那些内容都属于 `04-plot/`。

## 这里应该放什么

- 章节初稿
- 场景草稿
- 试写片段
- 成段可阅读的小说正文

## 这里不应该放什么

- “这一章要完成什么”的功能说明
- 角色关系备注
- 剧情节点清单
- 章节结构拆分

如果文档主要是写给作者自己看的，它通常不该放在这里。

## 当前建议结构

- `volume-01/`
  - 第一卷正文草稿

## 给协作者的阅读方法

先读本目录 README。

如果要看正式正文：

- 再进入 `volume-01/`
- 先读那个目录的 `README.md`
- 然后再打开具体章节草稿
