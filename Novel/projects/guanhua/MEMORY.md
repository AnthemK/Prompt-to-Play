## 已确定的工作原则

- 《观化》是中文小说，创作性文档默认以中文为准
- 世界观和设定严谨性是高优先级
- 角色塑造和角色弧光是高优先级
- 先完成结构和文档边界，再进入内容生产
- 大纲建立前，必须先明确世界规则与角色驱动力
- 必须持续维护文档的分层、分级、目录位置与互相索引
- 新增文档时，要优先判断它属于哪一层，而不是先写再塞
- 目录与索引一旦失真，后续设定会迅速失控，因此要主动收束
- 文档结构应尽量保持“树型分层 + 逐层展开 + 唯一出处”
- 项目顶层明确区分正式层与运行层：
  - 正式层用于长期维护
  - 运行层用于实验、缓存与当前任务支持
- README 不只是薄路标，还应概括本层目录整体内容
- 若存在下一级目录，本层 README 原则上只展开到下一层，并把阅读者继续交给下一层 README
- 同一项具体内容应只有一个权威落点，其他地方只做引用和指路
- 索引职责固定区分为四类：
  - README：目录入口
  - index：人工阅读索引
  - meta：结构化导航索引
  - cache：当前任务临时上下文包
- 文件与目录命名应尽量保持可判读、可检索、可扩展，并遵守统一命名规则

## 已明确的内容方向

- 主题方向：
  - 展示在一个真实的修仙世界中，人到底应该如何生存
- 主角方向：
  - 成长型主角
  - 修仙过程也是不断了解世界的过程
  - 最终指向修心
- 主角起点：
  - 宗门外门弟子
  - 出身平凡
  - 可适度配置金手指，但不能破坏世界真实感
  - 当前定名为“陆守一”
  - 当前年龄为十四岁
  - 当前修为为练气三层
  - 天赋方向为“内秀于心”
- 主角所属宗门：
  - 正派大宗门“太一门”
  - 整体确实在做正道之事
  - 内部存在坏人、污垢和不合理制度
  - 资源稀缺，内部同时存在良性与恶性竞争
- 世界冲突方向：
  - 主要反派不是纯粹邪恶宗门
  - 主要对手更偏政治斗争意义上的敌手
  - 可设置多个种族，并让其在天性或生存结构上存在利益对立
  - 种族差异应先来自客观生理条件与居住环境差异
  - 再合理衍生出文明结构、价值观与治理方式的差异
  - 各方未必“有错”，但矛盾会真实发生
  - 同一种族内部也存在不同利益集团与斗争
- 宗门核心目标：
  - 维持宗门存续并发扬光大
- 宗门最主要的正道职责：
  - 维系其统治下的人间秩序
- 世界运行优先级：
  - 资源真实
  - 秩序真实
  - 认知真实
  - 代价真实
- 社会架构分析方法：
  - 应优先借助现代政治学、社会学、经济学理论分析制度、资源与权力结构
  - 但不能机械照搬现实模型
  - 必须同时分析超凡力量对生产、治理、战争、知识、阶层与正当性的改写

  - 至少应存在宗门、修仙世家、国家政权 / 学宫三类稳定组织形态
  - 各类势力都必须回答资源、人才、愿景、责任与制度接口问题
  - 修仙世界中个人伟力可以在关键时刻压过制度框架
  - 宗门与世家、学宫之间必须同时体现合作与竞争

## 当前风险

- 如果过早堆设定，容易形成脱离剧情的百科化材料
- 如果过早写章纲，角色弧光和世界规则会被后补，导致返工
- 仙侠体系一旦规则不稳，后续战力、因果、境界和代价会持续失衡
- 如果“真实的修仙世界”没有被具体化，主题会停留在口号层
- 如果全卷一句话章纲不继续下钻为可写细纲，后续正文容易重新发散
- 如果角色弧光只存在于情节表、不回填人物档案，群像会逐渐失真
- 如果在卷一完整骨架刚闭合时立刻写正文，而不先做正文细纲，前中段仍可能因节奏密度不均而返工
- 如果第一章文气控制不稳，后续正文很容易滑回普通网文腔，必须及时校准
