# 03 Characters

这个目录处理角色系统，而不是角色百科。

核心问题：

- 这个角色在故事里起什么作用
- 他如何改变主角，或被主角改变
- 他的弧光如何展开
- 他在哪些阶段必须出现

## 这个目录解决什么问题

如果你是新协作者，这一层主要回答：

- 角色在故事里承担什么功能
- 角色之间的张力线在哪里
- 角色的基础设定是什么
- 经过卷一骨架后，哪些变化已经稳定

## 当前结构

### `01-indexes/`

放角色系统的总索引与总账。

包括：

- 角色总表
- 关系图
- 弧光总索引

### `meta/`

放角色系统的结构化导航索引。

这一层只解决机械问题，例如：

- 角色对应的权威 Markdown 在哪里
- 角色是独立立传还是合并收录
- 与哪些剧情文件存在直接弱连接

### `02-protagonist/`

只放主角陆守一的正式文档。

### `03-major-characters/`

放需要单独立传的主要角色。

当前标准：

- 主角之外，对主线有长期结构性影响的人物

### `04-casts/`

放卷级角色组与群像文档。

适合收：

- 外门群像
- 卷一关键角色组
- 地方接口角色组

## 归档原则

- 主要角色必须单独开文件立传
- 次要角色、同类角色可以按功能合并到组文档
- 一旦某个配角升格为主要角色，应及时从组文档拆出
- 剧情推进导致人物明显变化后，必须同步回填到对应角色文档

## 给协作者的阅读方法

建议先读本目录 README。

如果只想快速抓人：

- 先去 `meta/`
- 再去主角档
- 再去主要角色档
- 最后才看群像文档
