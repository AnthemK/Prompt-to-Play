# 04 Plot

这个目录处理长线结构和分层大纲。

## 这个目录解决什么问题

如果你是新协作者，这一层主要回答：

- 故事怎样展开
- 每一卷的功能是什么
- 每一阶段推进什么
- 章节为什么存在

这里是结构层，不是正文层。

顺序建议：

1. 先写长线蓝图
2. 再写分卷路线
3. 再拆章纲
4. 最后做伏笔回收账本

如果前面的世界规则和角色弧光还不稳，不建议直接开章纲。

## 给协作者的阅读方法

建议：

1. 先读本目录 README
2. 再进入 `volume-01/`
3. 在 `volume-01/` 里先读该目录 README
4. 再按任务需要打开卷一具体文件
