# Volume 01 Plot

这个目录专门放第一卷的大纲材料。

## 这个目录解决什么问题

它主要回答：

- 第一卷为什么这样排
- 五阶段怎么咬合
- 每一章在结构上承担什么功能
- 当前卷一已经细到什么程度

## 放什么

- 第一卷卷纲
- 第一卷阶段事件链
- 第一卷角色推进拆分
- 第一卷章纲

## 不放什么

- 世界设定正文
- 宗门制度设定
- 正文草稿

补充：

- “第一次下山任务模型”放在 `02-setting/`
- “主角第一次下山具体事件链”放在这里

## 当前建议文件

- `volume-01-outline.md`
  - 第一卷卷纲主文档
- `event-chain.md`
  - 第一卷阶段事件链
- `character-beats.md`
  - 第一卷角色推进节点
- `chapter-outline.md`
  - 第一卷章纲
- `volume-01-readiness.md`
  - 第一卷进入事件链前的准备度检查

## 给协作者的阅读建议

如果你只做卷一实验，建议顺序是：

1. 先读本目录 README
2. 再读 `volume-01-outline.md`
3. 再读 `volume-01-skeleton-review.md`
4. 任务偏结构时看 `chapter-outline.md`
5. 任务偏开篇落文时看：
   - `chapter-001-010-detail-outline.md`
   - `chapter-011-020-detail-outline.md`
   - `chapter-021-028-detail-outline.md`
