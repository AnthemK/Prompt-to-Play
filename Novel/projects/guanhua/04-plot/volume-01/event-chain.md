# Volume 01 Event Chain

## 用途

拆解第一卷的阶段事件链。

## 一、卷一主线判断

第一卷的核心，不应只是“外门逆袭”。

更准确地说，是：

- 一个相信宗门强权秩序总体必要的外门少年
- 在太一门的日常筛选、任务分配、下山实务与公开竞争中
- 一步步学会如何活、如何看、如何守住自己
- 最终在大比中以所有人都不能再忽视的方式显名

所以卷一真正要写的是四条线并行推进：

1. 生存线
2. 修炼线
3. 认知线
4. 显名线

还要补上一条隐藏标准：

5. 债务线

这里的“债务”不是只指金钱或人情，而是：

- 主角欠了谁
- 谁欠了主角
- 主角信过谁
- 主角被谁改变过
- 哪些关系会在后面反噬或成全他

除此之外，还应再加上一条执行要求：

- 每一阶段都要有明确的资源焦点与阶段代价

也就是说：

- 主角此时在争什么
- 他凭什么去争
- 他争到后会得到什么
- 他为此会失去什么或被谁盯上

卷一如果只完成“显名”，而没有立下这些情感债、秩序债和判断债，后续长线会发空。

## 二、卷一体量判断

用户暂定为：

- 约一百五十章

这个数字可以成立，但必须满足一个前提：

- 章节功能要足够细密
- 不能把宗门日常写成重复变体

更稳的理解方式不是“必须一百五十章”，而是：

- 卷一合理区间约为一百二十章到一百五十章

原因：

- 若少于一百二十章，外门生态、支线、下山、大比四段都容易显得过急
- 若高于一百五十章，第一卷很容易开始内耗，尤其会在外门日常和大比预热上失血

当前先按：

- 一百四十章到一百五十章的上限型体量

来搭链条。

## 三、五段式主事件链

### 第一段：外门立身

建议章数：

- 第 1 章至第 28 章

阶段任务：

- 建立太一门外门的真实手感
- 让陆守一的低位生存逻辑成立
- 立住同舍、同辈、执事、教习、师长的第一轮关系
- 埋下第一轮小支线与后续显名的因
- 让太一门首先以“完整社会”而不是“修炼副本”的方式成立

阶段核心事件：

1. 外门旧院日常建立
2. 晨点、轮值、补额、功录等制度压力落地
3. 陆守一在一次小摩擦中显出“稳而不软”的底色
4. 第一次季度校看或阶段性根基复核
5. 因一次任务分配或补录风波，被卷入同辈竞争

阶段成果：

- 读者相信这个外门真的会磨人
- 陆守一不是天降奇才，而是已经学会在秩序里求活的人
- 宋平、许应、赵六衡、马会川、韩崇度的位置全部落定
- 主角对太一门产生第一轮真实的归属、敬畏与提防
- 主角开始建立“账感”：什么值得争，什么必须忍，什么暂时不能碰

阶段结尾转折：

- 陆守一第一次被放到“可再看一眼”的范围

### 第二段：外门争位

建议章数：

- 第 29 章至第 58 章

阶段任务：

- 把外门竞争从生活摩擦升级为资源与名录竞争
- 让主角开始争到第一批真正有用的东西
- 让周既白与卷一中层关键角色逐步入场
- 让配角逐渐不只是“同辈角色”，而成为主角看世界的不同坐标

阶段核心事件：

1. 一至两条制度型支线展开
2. 陆守一通过苦劳、判断或临场补位争到第一批补额与更完整修炼时段
3. 明面热门开始出现，外门“前列”概念立起来
4. 裴照川、梁秋衡等中层人物进入视野
5. 周既白第一次真正注意陆守一，但尚未正式提携

阶段成果：

- 主角从“活下去”进入“开始争”
- 外门不只是生存场，也成了公开比较场
- 读者开始知道卷一终局的大比不是空降
- 主角开始欠下第一批真正会留到后面的关系债
- 第一批关键资源的成本、门槛和后果被明确看见

阶段结尾转折：

- 陆守一进入第一次重要下山的摸底或候选名单

### 第三段：青桑下山

建议章数：

- 第 59 章至第 88 章

阶段任务：

- 把视野从宗门内卷拉到山下秩序
- 用一次实务性下山让主角见到正道治理的真实面貌
- 形成卷一第一次真正有重量的认知冲击
- 让主角第一次发现“维持秩序”的代价并不只落在纸面规则上

阶段核心事件：

1. 任务录名与临时抽调
2. 主角随队前往青桑县
3. 药材供册、苗册或河运交接出现异常
4. 地方官、学宫、药行、宗门驻点口径彼此不完全重合
5. 事务问题升级为带有风险的异常事件
6. 陆守一在关键场面中既守令又不盲从，真正进入周既白视线

阶段成果：

- 主角第一次明白“维持秩序”包含抽取、核验、压服与妥协
- 同辈关系出现重组，有人更近，有人更敌
- 周既白对陆守一的评价从“可看”转为“可雕”
- 主角对太一门的理解从“强大且必要”推进到“必要，但并不天然正确”
- 主角真正意识到：每一份稳定背后都有人在付账

阶段结尾转折：

- 返回宗门后，陆守一得到第一次真正改变路径的法门或指点

### 第四段：归山蓄势

建议章数：

- 第 89 章至第 118 章

阶段任务：

- 消化下山后的认知后果与制度后果
- 让主角完成卷一最关键的一轮内在与战法升级
- 正式把大比推到前台
- 让他把见过的外部现实重新带回山门内部理解

阶段核心事件：

1. 青桑任务回录、记功、记过与观察变化
2. 周既白开始有限提点，传 `澄心观息法`
3. 陆守一逐渐形成适合自己的攻守节奏
4. 外门热门排序趋于清晰，裴照川成为最稳定对照
5. 顾清栩留下一次更清晰但仍不重的印象
6. 大比录名、初筛与大比前气氛全面升温

阶段成果：

- 陆守一从“会活”开始走向“能赢”
- 他的成长不再只体现在修为上，而体现在判断、心性和法门整合上
- 大比已不是终点提示牌，而是所有人都在准备的现实场
- 重要关系的轻重、远近、可信度开始第一次重排
- 大比所需的准备链被逐步补齐，而不是临场突然开出来

阶段结尾转折：

- 陆守一以不算最被看好的姿态正式进入大比主舞台

### 第五段：大比显名

建议章数：

- 第 119 章至第 150 章

阶段任务：

- 公开结算卷一前面积累的所有东西
- 用一场外门大比把主角从边缘人推成正式被看见的人
- 让“夺魁”既是战绩，也是认知与秩序位置的变化
- 让前文所有制度、关系、法门与判断都在公开场合被重新定义

阶段核心事件：

1. 录名、初筛、复观与正式轮次依次展开
2. 主角先以“稳”过关，而非一上来碾压
3. 在不同类型对手面前，陆守一的打法逐步完整显形
4. 同辈、执事、教习、峰脉旁观者对他的判断不断修正
5. 与明面热门的关键交锋成为秩序意义上的正面碰撞
6. 主角最终夺魁

阶段成果：

- 陆守一彻底打出名头
- 外门、承道院、行务房与部分峰脉都不能再把他当边缘人
- 卷一完成从“如何活”到“如何立住”的第一层跃迁
- 卷一结束时，他赢得的不只是一场比试，而是第一次拥有了被更高秩序认真对待的资格
- 夺魁带来的新代价与新风险必须同时显形，不能只留荣耀

阶段结尾转折：

- 大比夺魁不是结束一切，而是把他推入更高、更危险、更复杂的视野

## 四、卷一必须控制的三类节奏风险

### 1. 外门日常重复

解决方法：

- 每一段宗门日常都必须承担不同功能
- 前段写制度压迫
- 中段写资源竞争
- 后段写大比前的空气变化

### 2. 下山线喧宾夺主

解决方法：

- 下山必须是卷一中段的认知重锤
- 但不能把第一卷写成独立历练卷
- 它的价值在于反照山门，而不是抢走大比主线

### 3. 大比写成纯热血通关

解决方法：

- 大比要持续调用前文的人际、制度、法门和认知积累
- 不写成单纯的升级爽局

### 4. 只写显名，不写代价与后续债务

解决方法：

- 卷一每一阶段都要留下可回收的人际与判断后果
- 夺魁应带来更多视线、更多拉拢、更多敌意，而不是圆满封卷
- 读者要感觉到主角是被推上更复杂的棋盘，而不是完成了一个新手教程

## 五、接下来如何落文档

当前最合理的顺序是：

1. 先据此补卷纲主文档
2. 再拆角色推进节点
3. 最后按阶段拆章纲
