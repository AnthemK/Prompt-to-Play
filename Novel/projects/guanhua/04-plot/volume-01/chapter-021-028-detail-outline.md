# Volume 01 Chapter 021-028 Detail Outline

## 用途

这份文档是第一卷第 21 章到第 28 章的正文细纲。

它继续承接前两份细纲文档的层级，不写正文句子，只处理：

- 每章主要场景顺序
- 每章视角重心
- 每章必须揭示的信息
- 每章关系推进
- 每章结尾钩子

## 使用原则

### 1. “被看见”只能到轻量级

这一段最危险的错误，是写成：

- 周既白正式入场
- 主角一战扬名
- 师长当场赏识传法

这些都太早。

这一段最合适的力度只有：

- 做成了一件事
- 被高一层的人顺手记住
- 在旧院和外门近景里位置微微抬升

### 2. 小任务必须从旧院秩序自然长出来

不能突然空降一个看起来像主角专属舞台的机会。

更合理的方式是：

- 临时补位
- 应急跑差
- 某次小型内部比试或实际事务的空缺

### 3. 第一阶段的结尾要有“位置微变”，不能像完成一卷

读者应在第 28 章感到：

- 他总算被看了一眼
- 但他离真正往上走还很远

## 第二十一章

### 章节主功能

- 收束根基校看余波

### 场景顺序

1. 校看次日或当晚的回流议论
2. 韩崇度那句评价在不同人口里变成不同分量
3. 陆守一短暂松气，又立刻想到这还不够
4. 外门日常重新压回身上

### 本章视角重心

- 被记一笔，只是得到继续熬下去的凭据

### 必须揭示的信息

- 校看结果并没有直接改命
- 但陆守一第一次在修炼意义上不再完全属于“无名旧院弟子”

### 关系推进

- 韩崇度的存在感进一步坐实
- 林晚照作为同层参照被主角真正记上

### 章尾钩子

- 一件需要临时补位的小任务风声传来

## 第二十二章

### 章节主功能

- 把“小任务”挂进旧院日常

### 场景顺序

1. 外门某处临时缺人、名单有漏或有人出岔
2. 杜蘅、马会川或其他执事在名单上来回点人
3. 主角本不在优先名单里
4. 局面因为别人的退缩、推诿或失误逼近他

### 本章视角重心

- 机会在外门里常常并不像机会，更像一件会砸人的事

### 必须揭示的信息

- 陆守一不是主动抢位置
- 他被卷进去，才符合这个阶段的人物状态

### 关系推进

- 杜蘅与马会川对同一件事的处理方式可形成一点差别
- 宋平、许应、赵六衡对“要不要接”会有不同判断

### 章尾钩子

- 名字真落到陆守一头上

## 第二十三章

### 章节主功能

- 主角被卷入不占优的局面

### 场景顺序

1. 陆守一得知自己被点去补位
2. 他快速判断风险、分量和可能后果
3. 周围人有的觉得他倒霉，有的觉得他可以试一试
4. 主角决定接，但不是热血地接

### 本章视角重心

- 真正符合他性格的选择，是算清后仍然去接

### 必须揭示的信息

- 他依然怕出头
- 但也已经开始知道，完全不接事更危险

### 关系推进

- 宋平可能给一句偏稳的提醒
- 许应会记住这件事是否能做成
- 赵六衡更偏看笑话或等着看他出错

### 章尾钩子

- 任务真正开始，而且明显不顺手

## 第二十四章

### 章节主功能

- 让局面开始见真章

### 场景顺序

1. 小任务或临时差事实操展开
2. 主角发现事情并不像表面那样简单
3. 某位同辈或一线管理角色在关键处露出短板
4. 陆守一先稳住，再一点点找位置

### 本章视角重心

- 稳、忍、判断第一次需要同时出现

### 必须揭示的信息

- 陆守一的强不在抢，而在接住混乱后不乱
- 同辈里至少一人的短板要具体暴露

### 关系推进

- 某些原本看不上他的人开始改口半句
- 同舍或邻院里会多一个“他居然能接住”的印象

### 章尾钩子

- 局面还有最后一道最难处

## 第二十五章

### 章节主功能

- 让主角把事真正做成

### 场景顺序

1. 局面进入最关键一环
2. 主角靠稳、准、节奏感把事接到底
3. 明面上仍然不是漂亮大胜
4. 但结果明确落到“这事成了”

### 本章视角重心

- 这不是惊艳，而是可靠

### 必须揭示的信息

- 陆守一最先被别人看见的，不是锋芒，而是可用
- 这和周既白后续会注意他形成因果前提

### 关系推进

- 宋平会对他更多一层认同
- 许应会重新计算他未来的分量
- 马会川或相关执事也会把这个名字记住

### 章尾钩子

- 旁观者里有一道更高层的视线掠过

## 第二十六章

### 章节主功能

- 余波回流到更高层视线里

### 场景顺序

1. 事后回录、口风或随口议论传开
2. 周既白或其他更高一层观察者从旁带过
3. 他的反应必须很轻，只到“记住名字”
4. 陆守一自己并不清楚这一笔意味着什么

### 本章视角重心

- 真正会改变人命运的，常常是别人顺手记下的一笔

### 必须揭示的信息

- 周既白绝不能正式入场
- 这次只允许“被看一眼”

### 关系推进

- 主角与周既白之间第一次产生极轻的结构性联系

### 章尾钩子

- 旧院和同辈层的风向开始变一点

## 第二十七章

### 章节主功能

- 回收第一阶段各条小账

### 场景顺序

1. 舍内关系变化回收
2. 同辈判断变化回收
3. 执事口风变化回收
4. 主角重新回到自己的旧物、旧屋和修炼里

### 本章视角重心

- 位置的变化，先体现在别人说话和看你的方式变了

### 必须揭示的信息

- 陆守一的位置的确抬了一点
- 但他还远没有脱离旧院逻辑

### 关系推进

- 宋平更稳地站到“可以说话的人”
- 许应与主角的距离感进入重新计算阶段
- 赵六衡的态度可能从轻视转为提防

### 章尾钩子

- 与资源、补额、名录有关的下一轮风向露头

## 第二十八章

### 章节主功能

- 第一阶段正式收束，挂到第二阶段

### 场景顺序

1. 主角回看自己这段时日的得失
2. 外门空气里隐约显出新的资源竞争味道
3. 主角对“接下来不能只熬”形成更清楚判断
4. 以一笔不重但有势的收尾句，挂到卷一第二阶段

### 本章视角重心

- 他不再是完全无人注意的旧院弟子

### 必须揭示的信息

- 第一阶段完成的是“位置微变”
- 不是出头
- 更不是翻身

### 关系推进

- 与外门整体秩序的关系推进一格
- 与第二阶段“功录、补额、争位”的主线正式接通

### 章尾钩子

- 主角看见或听见一条和功录、补额、名录竞争直接相关的消息
- 第二阶段“外门争位”正式开启
