# Writing Execution Rules

## 用途

这份文档只处理《观化》在具体写作执行中的规则。

它回答的是：

- 落文时哪些问题必须主动检查
- 角色分布和行为逻辑要守哪些线
- 哪些执行偏差最容易把文本写坏

它不承担作品本体的思想说明；小说的文风原则与思想取向请读：

`01-foundations/novel-style-principles.md`

## 语言执行规则

### 1. 创作性文档默认以中文为准

《观化》是一部中文小说。  
因此，设定、人物、大纲、正文、修订意见与大多数创作性说明，默认都应直接使用中文撰写。

若确有需要引用外部理论、术语或方法，  
可以保留少量英文关键词；  
但英文不应反客为主，也不应破坏整体中文语境。

## 角色分布与行为逻辑要求

### 1. 男女角色比例应主动校正

《观化》允许因场景结构出现局部失衡。

例如：

- 宿舍
- 某些外门事务
- 某些峰脉环境

男角色与主角的互动可能天然更多。

但从整卷和整书层面看，重要男女角色比例应尽量维持在：

- 接近 `1:1`

这不是机械配额，而是为了避免叙事视野长期单侧化。

### 2. 男女角色都必须允许承担复杂功能

无论男女，都可以是：

- 正派
- 反派
- 中间派
- 被裹挟者
- 体制维护者
- 秩序挑战者

关键不在性别，而在：

- 其利益位置是什么
- 其认知边界是什么
- 其行为代价是什么

### 3. 不要把性别当成人物逻辑的捷径

不能因为角色是女性，就默认她：

- 更柔软
- 更善良
- 更适合承担感情功能

也不能因为角色是男性，就默认他：

- 更适合承担政治性
- 更适合承担权力性
- 更适合承担暴力与竞争

判断角色时，应优先问：

- 她或他在这个秩序里站在哪里
- 她或他想保住什么，又想争什么
- 她或他为什么会这样做

## 执行时的检查原则

### 1. 不要让思想压过现场

如果一段内容可以通过动作、关系和后果来成立，就不要先落成抽象议论。

### 2. 不要让语言替代结构

句子再漂亮，也不能替代：

- 关系推进
- 信息变化
- 局势变化
- 代价落地

### 3. 不要让设定说明接管正文

设定只能为人物和事件服务，不能在正文里变成大段说明书。

## 当前最重要的执行风险

- 容易因叙事惯性让男性承担大部分制度、竞争与政治功能，而女性只承担情感或点缀功能
- 容易把复杂政治与种族冲突偷换成简单道德审判
- 容易在正文中让作者判断压过人物判断
- 容易把设定说明、哲学判断和抒情句子写成正文推进的替代品
