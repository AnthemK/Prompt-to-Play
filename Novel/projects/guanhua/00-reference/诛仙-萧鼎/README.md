# 诛仙-萧鼎

这个目录放《诛仙》相关的参考材料。

它的用途不是保存“看过什么”，而是服务《观化》的实际创作问题：

- 宗门仙侠如何搭建世界入口
- 成长线如何和秩序线、感情线缠在一起
- 经典仙侠作品如何让设定真正服务剧情

## 本目录包含什么

- `诛仙（萧鼎著作的长篇仙侠小说）_百度百科.html`
  - 本地保存的原始网页
- `诛仙（萧鼎著作的长篇仙侠小说）_百度百科.md`
  - 从原始网页整理出的完整参考稿
- `《诛仙》 作者：萧鼎.txt`
  - 原文文本材料
- `guanhua-reference-notes.md`
  - 从《观化》项目角度提炼出的可用结论

## 建议阅读顺序

1. 先读 `guanhua-reference-notes.md`
2. 需要核对信息时，再读 `诛仙（萧鼎著作的长篇仙侠小说）_百度百科.md`
3. 需要核对文气、节奏或具体叙事感觉时，再看 `《诛仙》 作者：萧鼎.txt`
