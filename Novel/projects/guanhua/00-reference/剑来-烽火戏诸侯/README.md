# 剑来-烽火戏诸侯

这个目录放《剑来》相关的参考材料。

它的用途不是保存“看过什么”，而是服务《观化》的实际创作问题：

- 文字如何兼具文气、民间性与修行世界的辽阔感
- 大量人物、地理与设定并行时，如何仍维持长期叙事张力
- 少年成长、天下游历、群像与道义讨论如何彼此缠绕

同时需要明确：

- 《观化》借鉴的是这部作品在文气、游历感、群像厚度和人间气上的能力
- 不直接照搬它的超高文本体量、铺陈规模和某些过度散开的支线写法
- 《观化》仍要保证主角主线集中、设定节制、秩序逻辑清楚

## 本目录包含什么

- `剑来（烽火戏诸侯创作的网络仙侠小说）_百度百科.html`
  - 本地保存的原始网页
- `剑来（烽火戏诸侯创作的网络仙侠小说）_百度百科.md`
  - 从原始网页整理出的完整参考稿
- ` 烽火戏诸侯 - 剑来 - libgen.li.txt`
  - 原文文本材料
- `guanhua-reference-notes.md`
  - 从《观化》项目角度提炼出的可用结论

## 建议阅读顺序

1. 先读 `guanhua-reference-notes.md`
2. 需要核对世界结构、角色分布与卷次组织时，再读 `剑来（烽火戏诸侯创作的网络仙侠小说）_百度百科.md`
3. 需要核对原作文气、人物语感与游历段落时，再看 TXT 原文
