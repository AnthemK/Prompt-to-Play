# 00 Reference

这个目录放《观化》的外部参考与研究笔记。

这一层不直接产出正式设定，而是负责两件事：

1. 明确《观化》当前需要向哪些作品、理论或材料学习。
2. 把外部材料整理成可复用的项目经验，再决定哪些结论值得回写到正式层。

## 这一层解决什么问题

如果你要研究名作仙侠小说的结构经验、某类宗门叙事与成长路径、世界架构方法、文风气质来源，或者政治学、社会学、经济学、哲学等外部知识如何进入《观化》，应先来这里。

这里的文档应回答两个问题：

> 我们为什么读这份材料？
>
> 读完之后，哪些经验能真正服务《观化》？

## 文件索引

- `research-notes.md`
  - 本目录总索引。先看这里，确认当前有哪些参考材料，以及每份材料主要服务什么问题。
- `诛仙-萧鼎/`
  - 《诛仙》相关资料。重点看宗门仙侠的情感、氛围、师门关系与第一卷重心。
- `凡人修仙传-忘语/`
  - 《凡人修仙传》相关资料。重点看稀缺性、准备链、资源系统与成长真实感。
- `修真四万年-卧牛真人/`
  - 《修真四万年》相关资料。重点看大规模世界架构、制度分层与设定系统化能力。
- `一世之尊-爱潜水的乌贼/`
  - 《一世之尊》相关资料。重点看大设定如何向主角主线收束，以及人物与势力的“道路感”。
- `大道朝天-猫腻/`
  - 《大道朝天》相关资料。重点看宗门气、克制叙事、人物出场分量与静气。
- `剑来-烽火戏诸侯/`
  - 《剑来》相关资料。重点看人间气、群像辨识度、游历见闻与世界厚度。

如果要继续往下读，不要在这一层展开所有细节，而是进入对应子目录，再读该子目录自己的 `README.md`。

## 边界

这里不直接替代：

- `01-foundation/`
- `02-setting/`
- `03-characters/`
- `04-plot/`

研究笔记只能作为输入层。

若某条结论已经稳定，应回写到对应正式文档，而不是长期只留在参考层。

## 建议阅读方法

1. 先读本目录 README
2. 再读 `research-notes.md`
3. 按当前问题进入对应参考书子目录，并阅读该子目录的 `README.md`
4. 只在需要时继续下钻到整理稿、原始材料或项目内提炼笔记
5. 若结论已稳定，再回到正式层对应目录落档
