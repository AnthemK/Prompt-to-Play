# 大道朝天-猫腻

这个目录放《大道朝天》相关的参考材料。

它的用途不是保存“看过什么”，而是服务《观化》的实际创作问题：

- 宗门仙侠如何写得冷、稳、准，又不失人物温度
- 高度克制的叙事风格，如何与修行世界的深层设定结合
- 人物、宗门、天下格局如何在不喧哗的写法里慢慢显出重量

同时需要明确：

- 《观化》借鉴的是这部作品在文气、节制、人物出场控制和宗门气质上的能力
- 不直接照搬它的极端留白、极端淡化情绪或过于冷峻的距离感
- 《观化》仍要保留更充分的人物弧光、情感线和现实秩序描写

1. 先读 `guanhua-reference-notes.md`
2. 需要核对宗门结构、人物群像与条目整理时，再读 `大道朝天（猫腻创作的东方玄幻类网络小说）_百度百科.md`
