# 一世之尊-爱潜水的乌贼

这个目录放《一世之尊》相关的参考材料。

它的用途不是保存“看过什么”，而是服务《观化》的实际创作问题：

- 大型仙侠世界如何在高概念设定下仍然维持清楚层级
- 人物、势力、神话背景与修行体系如何被统一到同一叙事框架里
- 设定密度很高时，如何仍让核心人物和主线保持牵引力

同时需要明确：

- 《观化》借鉴的是这部作品在宏观结构、神话嫁接和主线收束上的能力
- 不直接照搬它的多元宇宙式规模、设定密度和时间回溯玩法
- 《观化》仍以宗门仙侠、修心成长和真实秩序为主轴
