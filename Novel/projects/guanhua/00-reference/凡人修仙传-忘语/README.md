# 凡人修仙传-忘语

这个目录放《凡人修仙传》相关的参考材料。

它的用途不是保存“看过什么”，而是服务《观化》的实际创作问题：

- 资源稀缺、准备成本与风险管理如何真正驱动修仙叙事
- 长篇升级线如何在多个大阶段里维持一致的生存逻辑
- 功法、丹药、法宝、材料与交易体系如何反复服务剧情
