# Heterodox And Forbidden Arts

这个目录处理本项目中**旁门术法与禁术**。

这里讨论的是“术”这一层的内容，  
也就是：修士在既有修行基础上，能够具体施展、运用、发动的偏门手段与高危手段。

## 这个目录解决什么问题

这一层主要处理以下内容：

- 什么样的术法会被归入旁门
- 什么样的术法会被归入禁术
- 旁门术法与禁术，和主流术法体系之间的边界是什么
- 为什么有些术法只是偏门，而有些术法会被修仙世界普遍视为不应存在
- 哪些具体旁门术法或禁术在当前设定中已经确定

## 边界

本目录负责：

- 旁门术法与禁术的总体定义
- 它们与主流术法体系之间的边界
- 修仙世界对禁术的基本态度
- 当前已经确定的具体旁门术法与禁术条目

本目录不负责：

- 修途与境界本身的定义
- 功法体系本身的定义
- 旁门功法与禁法的总体定义
- 正统术法、炼体手段、炼魂手段的正常主干体系
- 百艺、法宝、阵法、战争体系等外部支撑内容
- 具体人物最终掌握哪些旁门术法或禁术

换言之，本目录只讨论**术法层面的偏门与禁忌**，  
不讨论**功法层面的偏门与禁忌**。

## 术法与功法的区别

这一点必须明确。

- **功法**重在“修”，处理的是长期修炼路径、自身结构如何被改变、修士为何能成为什么样的人。
- **术法**重在“用”，处理的是修士如何施展、如何发动、如何把已有修为转化为具体效果。

因此：

- 旁门功法与禁法，属于**修炼路径**层的问题。
- 旁门术法与禁术，属于**具体手段**层的问题。

两者可能有关联，但不能混同。

## 分层与索引

**旁门术法与禁术总纲**

定义本目录下两类对象的基本边界。  
重点处理：

- 何谓旁门术法
- 何谓禁术
- 它们如何与主流术法区分
- 它们与“旁门功法 / 禁法”之间的关系与区别
- 两者之间是否重合、如何重合

### `[具体术法条目]`
**具体旁门术法或禁术**

后续凡已正式确定的具体条目，均直接放在本目录下，以中文名命名。  
例如：

- `某某秘术.md`
- `某某禁咒.md`

每一条具体条目，主要回答：

- 这门术为何被归入本目录
- 它依托什么修行基础
- 它如何施展
- 有什么代价、风险与边界
- 它为何受到排斥、警惕、封禁或围剿

## 建议阅读顺序

建议按以下顺序阅读：

1. `heterodox-and-forbidden-arts.md`
2. 本目录下已确定的具体术法条目

先看总纲，再看具体条目。  
这样可以先明确“术法层面的旁门与禁术”是什么，再理解具体内容。

## 使用原则

本目录当前阶段的目标是：

- 先把旁门术法与禁术的目录接口立住
- 先确保后续新增特殊术法时有明确归类位置
- 先为那些无法归入主流术法体系、或本身具有强烈副作用与社会危害的术法预留位置
- 等真正需要时，再逐步补入具体条目

因此，本目录允许保留以下接口：

- 后续新增且无法被主流术法体系吸收的特殊术法
- 后续被正式列为禁术的术法
- 某些从旁门术法滑入禁术的边缘类型
- 某些表面上并不邪恶、但后果极重而被列禁的特殊体系

但所有后续补充，都应服从本目录已经确定的边界与命名方式，  
不应在具体条目中混淆“术法”与“功法”这两个层级。
