# Mystic Arts

这个目录处理本项目中**术法、法门、秘术与具体运用手段**。

这里讨论的不是修行体系本身，也不是功法如何建立修士的长期修炼路径，  
而是：修士在既有修行基础上，究竟能够学会、掌握并施展哪些具体的“术”。

这里的“术”，包括但不限于：

- 法术
- 身法
- 遁法
- 剑诀
- 印法
- 咒法
- 护身法
- 杀伐法
- 秘术
- 各条修途在实战与现实运用中的具体手段

## 这个目录解决什么问题

这一层主要处理以下内容：

- 术法与功法的关系是什么
- 什么样的内容应被视为“术”，而不是“功法”或“法宝”
- 不同修途通常会发展出怎样的术法体系
- 正统、炼体、炼魂等不同路数的术法在风格上有何差异
- 哪些具体术法在当前设定中已经确定

## 边界

本目录负责：

- 术法体系的总体定义
- 术法与功法、法宝、百艺之间的边界
- 不同修途术法体系的总体画像
- 当前已经确定的具体术法条目

本目录不负责：

- 修途与境界本身的定义
- 功法体系的定义
- 百艺、杂术、炼丹、制符、布阵、炼器等技艺体系
- 法宝、异宝、本命物等器物体系
- 具体人物最终掌握了哪些术法
- 具体战役、战局与战争组织方式

换言之，本目录只讨论“术法本身是什么样的”，  
不直接展开“人物如何使用这些术法完成某场战斗”。

## 分层与索引

### `art-system.md`
**术法体系总纲**

定义“术”这一层在整个修仙体系中的位置。  
重点处理：

- 何谓术法
- 术法与功法的区别
- 术法与法宝、百艺的区别
- 一门术通常如何成立
- 术法在修行者成长中的作用

### `orthodox-arts/`
**正统术法**

处理正统修道一路衍生出的术法体系。  
重点会包括：

- 正统术法的总体风格
- 正统术法与正统功法的关系
- 正统术法在攻、防、御、行、控等方面的基本展开

### `body-arts/`
**炼体手段**

处理炼体一路衍生出的具体手段。  
重点会包括：

- 炼体者的搏杀法
- 身法、冲阵、破坚、护体、续战等具体手段
- 炼体手段为何通常更直接、更强硬、更依赖本体

### `soul-arts/`
**炼魂手段**

处理炼魂一路衍生出的具体手段。  
重点会包括：

- 幻惑、摄心、守神、识伪、动念等方向
- 炼魂手段为何强于对人、弱于对物
- 炼魂手段与神魂稳定、妄念风险之间的关系

### `heterodox-and-forbidden-arts/`
**旁门术法与禁术**

处理那些无法被主流术法体系完整吸收，  
或本身具有巨大副作用与外部危害的术法。

## 建议阅读顺序

建议按以下顺序阅读：

1. `art-system.md`
2. 各修途对应的术法总纲
3. 后续具体术法条目

先看总纲，再看分类，再看具体条目。  
这样可以先明确“术法是什么”，再理解不同路数究竟如何展开。

## 使用原则

本目录当前阶段的目标是：

- 先把“术法”这一层从功法、法宝、百艺中清楚分离出来
- 先立住不同修途的术法风格边界
- 先为后续第一卷可能会实际用到的具体术法预留位置
- 具体条目只在真正需要时再逐步补入

因此，本目录允许保留以下接口：

- 后续新增的具体术法条目
- 某些与具体功法强绑定的核心手段
- 某些极偏门、极高危或极具代表性的秘术与禁术
- 某些在不同修途中都会出现、但展开方式不同的共通术法类型

但所有后续补充，都应服从本目录已经确定的边界与命名方式，  
不应在具体条目中反向改写“术、功法、法宝、百艺”之间的层级关系。
