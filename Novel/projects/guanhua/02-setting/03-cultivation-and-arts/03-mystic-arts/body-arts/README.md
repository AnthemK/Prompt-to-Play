# Body Arts

这个目录处理本项目中**炼体一路所衍生出的具体手段**。

这里讨论的不是炼体修途本身，也不是炼体功法如何长期塑造肉身，  
而是：炼体者在既有体魄、气血、承载与内景基础上，究竟能够发展出怎样的具体“术”。

这里的“炼体手段”，包括但不限于：

- 搏杀法
- 破坚法
- 护体法
- 冲阵法
- 身法
- 近身镇压之法
- 爆发法
- 续战法
- 与高阶肉身异象、内景秩序相关的外显手段

## 这个目录解决什么问题

这一层主要处理以下内容：

- 炼体一路会发展出怎样的具体手段
- 炼体手段与炼体功法是什么关系
- 炼体手段为何通常更直接、更强硬、更依赖本体
- 炼体手段在攻、防、行、破、承、续等方面通常如何展开
- 哪些具体炼体手段在当前设定中已经确定

## 边界

本目录负责：

- 炼体手段的总体画像
- 炼体手段与炼体功法之间的关系
- 炼体手段内部的主要展开方向
- 当前已经确定的具体炼体手段条目

本目录不负责：

- 炼体修途本身的定义
- 炼体功法体系本身的定义
- 正统术法、炼魂手段、旁门禁术体系
- 百艺、法宝、阵法、战争体系等外部支撑内容
- 具体人物最终掌握哪些炼体手段

换言之，本目录只讨论“炼体手段是什么样的”，  
不直接展开“某个角色如何把这些手段组合成完整战斗风格”。

## 分层与索引

### `body-arts.md`
**炼体手段总纲**

定义炼体手段这一大类的基本边界。  
重点处理：

- 炼体手段的总体风格
- 炼体手段与炼体功法的关系
- 炼体手段为何通常更重本体、更重承压、更重近身落实
- 炼体手段在攻、防、行、破、承、续等方向上的基本展开

### `[具体手段条目]`
**具体炼体手段**

后续凡已正式确定的炼体手段，均直接放在本目录下，以中文名命名。  
例如：

- `某某镇岳拳.md`
- `某某护体法.md`
- `某某踏虚步.md`

每一条具体条目，主要回答：

- 这门手段修什么、用什么
- 如何施展
- 适合什么样的炼体者
- 与何种炼体功法或分支关系最深
- 有什么代价、边界与典型用途

## 建议阅读顺序

建议按以下顺序阅读：

1. `body-arts.md`
2. 本目录下已确定的具体炼体手段条目

先看总纲，再看具体条目。  
这样可以先明确“什么才算炼体手段”，再理解不同类型手段的差异。

## 使用原则

本目录当前阶段的目标是：

- 先把炼体手段的整体画像立住
- 先确立炼体手段与炼体功法之间的关系
- 先为后续第一卷可能出现的关键炼体手段预留接口
- 具体条目只在真正需要时再逐步补入

因此，本目录允许保留以下接口：

- 不同炼体分支衍生出的手段差异
- 与某部炼体功法强绑定的核心手段
- 同一手段在不同肉身层次下的不同表现
- 某些在修仙世界中流传较广的通用炼体技法

但所有后续补充，都应服从本目录已经确定的边界与命名方式，  
不应在具体条目中反向改写“炼体手段”这一层的总体定义。
