# Orthodox Arts

这个目录处理本项目中**正统修道一路所衍生出的术法**。

这里讨论的不是正统修道的功法体系本身，  
也不是正统修士在长期修行中如何推进境界，  
而是：在正统修道的根基之上，修士通常会发展出怎样的具体术法与运用手段。

正统术法的根本特点，不在于极端偏锋，  
也不在于单点怪异，  
而在于：

- 体系完整
- 展开成熟
- 手段丰富
- 阴阳五行、灵力流转与整体统摄能力强
- 在攻、防、御、行、控等方面通常最为均衡

因此，正统术法往往是修仙世界中最常见、最成熟、也最具“正法”气质的一类术法体系。

## 这个目录解决什么问题

这一层主要处理以下内容：

- 正统术法的总体风格是什么
- 正统术法为何通常最完整、最多样、最成熟
- 正统术法与正统功法之间是什么关系
- 正统术法通常会沿哪些方向展开
- 哪些具体正统术法在当前设定中已经确定

## 边界

本目录负责：

- 正统术法的总体画像
- 正统术法与正统功法之间的关系
- 正统术法内部的主要展开方向
- 当前已经确定的具体正统术法条目

本目录不负责：

- 正统修道这条修途本身的定义
- 正统功法体系本身的定义
- 炼体、炼魂、旁门与禁术体系
- 法宝、百艺、阵地工程、战争体系等外部支撑内容
- 具体人物最终掌握哪些正统术法

换言之，本目录只讨论“正统术法是什么样的”，  
不直接展开“某个角色如何把这些术法组合起来”。

## 分层与索引

### `orthodox-arts.md`
**正统术法总纲**

定义正统术法这一大类的基本边界。  
重点处理：

- 正统术法的总体风格
- 正统术法与正统功法的关系
- 正统术法为何通常手段最多、体系最成熟
- 正统术法在攻、防、御、行、控等方向上的基本展开

### `[具体术法条目]`
**具体正统术法**

后续凡已正式确定的正统术法，均直接放在本目录下，以中文名命名。  
例如：

- `某某剑诀.md`
- `某某遁法.md`
- `某某护身印.md`

每一条具体条目，主要回答：

- 这门术法修什么、用什么
- 如何施展
- 适合什么样的修士
- 与何种功法或法脉关系最深
- 有什么代价、边界与典型用途

## 建议阅读顺序

建议按以下顺序阅读：

1. `orthodox-arts.md`
2. 本目录下已确定的具体正统术法条目

先看总纲，再看具体条目。  
这样可以先明确“什么才算正统术法”，再理解各类具体手段的差异。

## 使用原则

本目录当前阶段的目标是：

- 先把正统术法的整体画像立住
- 先确立正统术法与正统功法之间的关系
- 先为后续第一卷可能出现的关键正统术法预留接口
- 具体条目只在真正需要时再逐步补入

因此，本目录允许保留以下接口：

- 不同法脉衍生出的术法差异
- 与某部正统根本法强绑定的核心术法
- 同一术类在不同正统流派中的不同展开
- 某些在修仙世界中广泛流传的正统通用法门

但所有后续补充，都应服从本目录已经确定的边界与命名方式，  
不应在具体条目中反向改写“正统术法”这一层的总体定义。
