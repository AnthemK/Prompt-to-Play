# Esoteric Crafts

这个目录处理本项目中修仙世界里的**百艺与杂术体系**。

这里讨论的不是功法，也不是直接用于对敌的术法，  
而是那些更偏向于长期准备、条件塑造、资源转化、器物制造、环境经营与状态维护的技艺。

## 这个目录解决什么问题

这一层主要处理以下内容：

- 百艺是什么
- 阵法、炼器、炼丹、医养、灵兽五门分别处理什么
- 哪些内容应归入百艺，而不应归入术法或法宝体系
- 阅读这一部分时，应当如何把握五门百艺之间的边界

## 边界

本目录负责：

- 百艺体系的总体定义
- 阵法、炼器、炼丹、医养、灵兽五门的基本分工
- 当前已经确定的具体百艺设定条目

本目录不负责：

- 修途与境界本身的定义
- 功法体系本身的定义
- 直接战斗用术法体系
- 法宝成品体系本身的分类与品级
- 具体人物学习哪一门副职业
- 具体剧情中的工坊、堂口、商路与势力运作细节

换言之，本目录讨论的是：

> **修仙世界中的百艺是什么。**

而不是：

> **修士当场怎么出手，或某件法宝本身如何分类。**

## 分层与索引

### `craft-system.md`
**百艺总纲**

适合先读。  
用于把握百艺这一层的总体边界，以及五门百艺的基本分工。

### `formations.md`
**阵法**

本目录中最重要、也最需要详写的一门。  
如果想先理解百艺如何与组织、环境、护持体系结合，可以优先阅读本文件。

### `artifact-forging.md`
**炼器**

处理“器物如何被炼成”这一层的问题。  
适合在阅读阵法之后继续阅读，用来理解炼器与法宝成品体系的边界。

### `alchemy.md`
**炼丹**

处理丹药如何被炼成，以及炼丹与其他百艺的关系。  
适合在炼器之后阅读。

### `medicine-and-nurture.md`
**医养**

处理活体状态的诊、治、调、养。  
建议与 `alchemy.md` 连续阅读，以把握两者边界。

### `spirit-beast-arts.md`
**灵兽**

处理灵兽在修仙世界中的使用体系。  
当前阶段只展开“如何使用”，不展开完整驯养学。

## 建议阅读顺序

建议按以下顺序阅读：

1. `craft-system.md`
2. `formations.md`
3. `artifact-forging.md`
4. `alchemy.md`
5. `medicine-and-nurture.md`
6. `spirit-beast-arts.md`

先看总纲，再看阵法，再依次看其余门类。  
这样可以先立住“百艺是什么”，再进入具体分类。

## 使用方式

阅读本目录时，建议始终注意三点：

1. 先分清百艺与功法、术法、法宝的边界  
2. 再看五门百艺各自处理什么  
3. 最后再进入具体门类内部的细分设定
