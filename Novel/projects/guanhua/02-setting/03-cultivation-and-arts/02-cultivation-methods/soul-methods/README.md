这个目录处理本项目中**炼魂 / 炼神一系的功法**。

这里讨论的不是“炼魂路线在战斗中如何出手”，  
也不是“幻术、控魂、感知、推演等具体手段如何展开”，  
而是：在炼魂路线中，功法通常以何种方式成立、分化、传承，并最终塑造出怎样的修士。

## 这个目录解决什么问题

这一层主要处理以下内容：

- 炼魂功法的共同特征是什么
- 炼魂功法为何最强调神魂稳定、感知能力、自我统御与高层接口
- 炼魂功法如何处理感知、误导、控制、风险与反噬之间的关系
- 炼魂功法内部通常会形成怎样的分化
- 哪些具体炼魂功法在当前设定中已经确定

## 边界

本目录负责：

- 炼魂功法的总体特征
- 炼魂功法与神魂、感知、自我统御、污染与反噬之间的关系
- 炼魂功法内部的典型分类
- 当前已经确定的具体炼魂功法条目

本目录不负责：

- 功法体系的总体定义
- 单个功法的一般内部结构模板
- 正统功法、炼体功法、旁门功法与禁法
- 战斗法门、招式、斗法流程与克制关系
- 幻术、控魂、侦测、天机、因果等具体手段的详细战斗展开
- 法宝、百艺、丹药、符箓、阵法等配套体系的细节
- 具体人物最终修炼哪一门功法

换言之，本目录只讨论“炼魂功法是什么样的”，不直接展开“炼魂修士具体怎么出手”或“谁学了哪门功法”。

## 使用原则

本目录当前阶段的目标是：

- 先把炼魂功法的整体画像立住
- 先整理已经讨论出的核心共识与主要分歧
- 先为前几卷可能涉及的炼魂路线预留接口
- 在真正需要时，再逐步补完具体功法、法脉与关键能力设定

因此，本目录允许保留以下接口：

- 不同炼魂路线的后续补充
- 偏感知、偏控魂、偏守心、偏推演、偏幻惑等不同方向的炼魂功法扩展
- 与特殊神魂资质、特殊媒介、特殊环境绑定的炼魂功法条目
- 高阶炼魂根本法与低阶通用炼魂法之间的层次补全

但所有后续补充，都应服从本目录已经确定的分类、边界与命名方式，不应在具体条目中反向改写本层结构。  
若后续写作需要超出本目录当前承载范围的炼魂设定，应先补设定，再继续创作。
