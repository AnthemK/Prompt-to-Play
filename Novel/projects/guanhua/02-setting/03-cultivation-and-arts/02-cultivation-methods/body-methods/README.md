# Body Methods

这个目录处理本项目中**炼体一系的功法**。

这里讨论的不是“炼体是什么”，也不是“炼体修士在战斗中如何出手”，  
而是：在炼体路线中，功法通常以何种方式成立、分化、传承，并最终塑造出怎样的修士。

## 这个目录解决什么问题

这一层主要处理以下内容：

- 炼体功法的共同特征是什么
- 炼体功法为何最强调肉身承载、生机维持与长期淬炼
- 炼体功法如何处理强度、损耗、修复与极限之间的关系
- 炼体功法内部通常会形成怎样的分化
- 哪些具体炼体功法在当前设定中已经确定

## 边界

本目录负责：

- 炼体功法的总体特征
- 炼体功法与肉身、气血、生机、损耗之间的关系
- 炼体功法内部的典型分类
- 当前已经确定的具体炼体功法条目

本目录不负责：

- 功法体系的总体定义
- 单个功法的一般内部结构模板
- 正统功法、炼魂/炼神功法、旁门功法与禁法
- 战斗法门、招式、斗法流程与克制关系
- 法宝、百艺、丹药、符箓、阵法等配套体系的细节
- 具体人物最终修炼哪一门功法

换言之，本目录只讨论“炼体功法是什么样的”，不直接展开“炼体修士怎么打”或“谁学了哪门功法”。

## 分层与索引

### `body-methods.md`
**炼体功法总纲**

定义炼体功法这一大类的共同特征。  
重点处理：

- 肉身承载
- 气血、生机与损耗
- 炼体功法的优势、限制与代价
- 炼体功法内部的基本分化
- 炼体功法为何高下限、强主战、但功能性较弱

### `[具体功法条目]`
**具体炼体功法**

后续凡已正式确定的具体炼体功法，均直接放在本目录下，以中文功法名命名。  
例如：

- `金身锻骨法.md`
- `龙象镇岳功.md`

每一条具体功法，主要回答：

- 这门功法修什么
- 如何修
- 适合谁修
- 会把修士塑造成什么样
- 有什么代价、风险与边界
- 它在炼体功法谱系中的位置是什么

## 建议阅读顺序

建议按以下顺序阅读：

1. `body-methods.md`
2. 本目录下已确定的具体功法条目

先看总纲，再看具体功法。  
这样可以先明确“什么样的功法才算炼体功法”，再理解每一门具体功法的风格与位置。

## 使用原则

本目录当前阶段的目标是：

- 先把炼体功法的整体画像立住
- 先确定第一卷会实际用到、或必须提前存在的关键炼体功法
- 其余功法条目先留白，后续按写作需要逐步补入

因此，本目录允许保留以下接口：

- 不同炼体路线的后续补充
- 偏重骨、血、皮、筋、脏腑等不同方向的炼体功法扩展
- 与特殊体质、特殊资源、特殊环境绑定的炼体功法条目
- 高阶炼体根本法与低阶通用炼体法之间的层次补全

但所有后续补充，都应服从本目录已经确定的分类、边界与命名方式，不应在具体条目中反向改写本层结构。
