# 02 Cultivation Methods

这个目录处理本项目中的功法与传承。

它主要回答的是：修行者具体如何沿某一路径推进，功法如何塑造修士的承载结构、资源需求、突破风险与手段风格，以及功法为何会成为宗门、世家与法统体系中的核心资产。

## 这个目录解决什么问题

这一层主要处理以下内容：

- 功法在修行体系中的地位是什么
- 一门功法通常由哪些部分构成
- 正统修道、炼体、炼魂三条主路分别会形成怎样的功法结构
- 多路同修相关功法为何稀少、危险且难以复制
- 功法为何会与传承、秘传、法脉控制与断代问题绑定

## 边界

本目录负责：

- 功法的地位与分类
- 功法的结构与组成
- 不同修途的典型功法形态
- 功法之间的相性、冲突与兼容问题
- 传承、秘传、残卷、禁法与失传问题

本目录不负责：

- 修途与境界的基础定义
- 斗法风格、战斗流程与术法分类
- 丹、符、阵、器、医护、御兽等百艺细节
- 法宝、本命法宝、重器与战略法宝体系
- 具体人物最终学了哪门功法、如何在剧情中得到该功法

换言之，本目录只定义“功法是什么、如何成立、如何分类、如何传承”，不直接展开“怎么打”“怎么做器物”“谁来使用”。

## 分层与索引

### `method-system.md`
**功法体系**

定义功法在整个修行体系中的位置。  
回答功法为何不是附属技巧，而是修行骨架的一部分。

### `method-structure.md`
**功法结构**

定义一门功法通常由哪些部分组成。  
回答功法如何分别处理引入、运转、塑形、突破、配套法门与资源依赖等问题。

### `orthodox-methods.md`
**修道功法**

整理正统修道路径下的功法形态。  
重点处理其“中轴、全面、体系最成熟、可接入手段最多”的特征。

### `body-methods.md`
**炼体功法**

整理炼体路径下的功法形态。  
重点处理其“高下限、强主战、低功能、重承压与持续作战”的特征。

### `soul-methods.md`
**炼魂功法**

整理炼魂路径下的功法形态。  
重点处理其“高上限、低下限、重布局、重误导、重高层接口”的特征。

### `hybrid-and-special-methods.md`
**兼修、特殊与禁忌功法**

整理双修、三修、偏门、残缺、失传与禁忌类功法。  
重点处理其高风险、高代价、难统合与不可普及的问题。

### `inheritance-and-secrecy.md`
**传承与秘传**

定义功法为何会成为宗门、世家与法统控制的核心资产。  
回答完整传承、残卷、秘传、断代、失传与选择性传授等问题。

## 建议阅读顺序

建议按以下顺序阅读：

1. `method-system.md`
2. `method-structure.md`
3. `orthodox-methods.md`
4. `body-methods.md`
5. `soul-methods.md`
6. `hybrid-and-special-methods.md`
7. `inheritance-and-secrecy.md`

这个顺序的目的，是先明确功法在修行体系中的位置，再明确功法本身的结构；  
在此之后，再分别进入三条主路与特殊功法，最后收束到传承与秘传问题。

## 使用原则

本目录先立功法体系的骨架，不追求一次性写尽所有具体功法。  
当前阶段的目标是：

- 先明确功法的地位、结构与分类
- 先确定第一卷必然会用到的关键功法类型
- 其余具体条目先留白，后续按写作需要逐步补入

因此，本目录允许保留以下接口：

- 具体根本法、辅法与专门法的后续增补
- 某些宗门独门功法与断代功法的后续补充
- 三修真正可行的高阶功法线索
- 与法宝、法门、百艺深度耦合的特殊功法

但所有后续补充，都应服从本目录已经确定的分类、边界与命名方式，不应在下层文本中反向改写本层结构。
