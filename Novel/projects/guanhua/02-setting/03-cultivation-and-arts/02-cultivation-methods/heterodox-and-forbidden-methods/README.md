# Heterodox And Forbidden Methods

这个目录处理本项目中**旁门功法与禁法**。

这里讨论的不是某一门具体功法如何运转，  
也不是战斗中会出现哪些具体邪门手段，  
而是：在整个功法体系之外，哪些功法无法被主流分类稳定吸收，哪些功法会进一步被修仙世界视为不应存在。

## 这个目录解决什么问题

这一层主要处理以下内容：

- 旁门功法与禁法各自是什么
- 两者与主流功法体系之间的边界是什么
- 为什么有些功法只是偏门，而有些功法会被进一步列为禁法
- 哪些具体旁门功法或禁法在当前设定中已经确定

## 边界

本目录负责：

- 旁门功法与禁法的总体定义
- 两者与主流功法体系之间的边界
- 修仙世界对禁法的基本态度
- 当前已经确定的具体旁门功法与禁法条目

本目录不负责：

- 功法体系的总体定义
- 单个功法的一般内部结构模板
- 已能被主流类别完整吸收的功法
- 具体战斗法门、招式、斗法流程与克制关系
- 法宝、百艺、丹药、符箓、阵法等配套体系的细节
- 具体人物最终修炼哪一门旁门功法或禁法

换言之，本目录只讨论“为什么某类功法被归入旁门或禁法”，  
不直接展开“这门法具体怎么施展”或“谁学了它”。

## 分层与索引

### `heterodox-and-forbidden-methods.md`
**旁门功法与禁法总纲**

用于定义本目录下两类对象的基本边界。  
重点处理：

- 何谓旁门功法
- 何谓禁法
- 两者为何成立
- 两者如何与主流功法区分
- 两者之间是否重合、如何重合

### `[具体功法条目]`
**具体旁门功法或禁法**

后续凡已正式确定的具体条目，均直接放在本目录下，以中文名命名。  
例如：

- `某某秘法.md`
- `某某禁术.md`

每一条具体条目，主要回答：

- 这门功法为何被归入本目录
- 它修什么
- 如何修
- 有什么代价、风险与边界
- 它在修仙世界中为何受到排斥、警惕、封禁或围剿

## 建议阅读顺序

建议按以下顺序阅读：

1. `heterodox-and-forbidden-methods.md`
2. 本目录下已确定的具体功法条目

先看总纲，再看具体条目。  
这样可以先明确“为什么某门法会落到这一类”，再理解其具体内容。

## 使用原则

本目录当前阶段的目标是：

- 先把“旁门功法”与“禁法”的目录接口立住
- 先确保后续新增功法时有明确归类位置
- 先为那些无法归入主流体系、或本身具有强烈副作用与社会危害性的功法预留位置
- 等真正需要时，再逐步补入具体条目

因此，本目录允许保留以下接口：

- 后续新增且无法被主流体系吸收的特殊功法
- 后续被正式列为禁法的功法
- 某些从旁门滑入禁法的边缘类型
- 某些表面上并不邪恶、但后果极重而被列禁的特殊体系

但所有后续补充，都应服从本目录已经确定的边界与命名方式，不应在具体条目中反向改写本层结构。
