# Orthodox Methods

这个目录处理本项目中**正统修道一系的功法**。

这里讨论的不是“修道是什么”，也不是“修仙六境如何划分”，  
而是：在正统修道路线中，功法通常以何种方式成立、分化、传承，并最终塑造出怎样的修士。

## 这个目录解决什么问题

这一层主要处理以下内容：

- 正统功法的共同特征是什么
- 正统功法为何最强调天人感应、调和中庸与天人合一
- 正统功法如何围绕灵气、灵力与属性体系运转
- 宗门公法、法脉解释与具体修炼版本之间是什么关系
- 正统功法内部会形成怎样的类型分化
- 哪些具体正统功法在当前设定中已经确定

## 边界

本目录负责：

- 正统功法的总体特征
- 正统功法与五行、阴阳、十干属性的关系
- 宗门公法与法脉解释的基本关系
- 正统功法内部的典型分类
- 当前已经确定的具体正统功法条目

本目录不负责：

- 功法体系的总体定义
- 单个功法的一般内部结构模板
- 炼体功法、炼魂/炼神功法、旁门功法与禁法
- 战斗法门、招式、斗法流程与克制关系
- 法宝、百艺、丹药、符箓、阵法等配套体系的细节
- 具体人物最终修炼哪一门功法

换言之，本目录只讨论“正统功法是什么样的”，不直接展开“正统修士如何斗法”或“谁学了哪门功法”。

## 分层与索引

### `orthodox-methods.md`
**正统功法总纲**

定义正统功法这一大类的共同特征。  
重点处理：

- 天人感应
- 调和中庸
- 天人合一
- 灵气与灵力的组织
- 属性体系在不同修行阶段中的地位
- 宗门公法与法脉解释的关系

### `[具体功法条目]`
**具体正统功法**

后续凡已正式确定的正统功法，均直接放在本目录下，以中文功法名命名。  
例如：

- `长春功.md`
- `七杀剑诀.md`
- `长生六道轮回经.md`

每一条具体功法，主要回答：

- 这门功法修什么
- 如何修
- 适合谁修
- 会把修士塑造成什么样
- 有什么代价、风险与边界
- 它在正统功法谱系中的位置是什么

## 建议阅读顺序

建议按以下顺序阅读：

1. `orthodox-methods.md`
2. 本目录下已确定的具体功法条目

先看总纲，再看具体功法。  
这样可以先明确“什么样的功法才算正统功法”，再理解每一门具体功法的风格与位置。

## 使用原则

本目录当前阶段的目标是：

- 先把正统功法的整体画像立住
- 先确定第一卷会实际用到、或必须提前存在的关键正统功法
- 其余功法条目先留白，后续按写作需要逐步补入

因此，本目录允许保留以下接口：

- 不同宗门公法的后续补充
- 同源法脉不同解释版本的后续扩展
- 属性倾向不同的正统功法条目
- 高阶正统根本法与低阶通用法之间的层次补全

但所有后续补充，都应服从本目录已经确定的分类、边界与命名方式，不应在具体条目中反向改写本层结构。
