# Treasures And Artifacts

这个目录处理本项目中修仙世界里的**宝物体系**。

这里讨论的不只是法宝，  
也包括那些本身就足以被争夺、封存、传承、炼化或作为组织底蕴的珍稀宝物。

## 这个目录讲什么

这一层主要处理：

- 宝物本身是什么
- 宝物大致可以如何分类
- 宝物的品级、品质、契合、认主与温养应如何理解
- 不同类型宝物在个人与组织层面分别有什么意义

## 这个目录不讲什么

这一层不处理：

- 宝物如何被炼制出来的具体流程
- 炼器、炼丹、阵法等百艺本身的技术体系
- 具体术法如何借助宝物施展
- 修士的修途、境界与功法体系
- 具体人物持有哪件宝物的剧情安排

换言之，本目录讨论的是：

> **宝物本身是什么。**

而不是：

> **它是如何被造出来的，或修士拿着它具体怎么施术。**

## 文件索引

### `treasure-system.md`
**宝物总纲**

适合先读。  
用于把握这一层的总体边界，并理解什么叫“宝物体系”。

### `grades-and-compatibility.md`
**品级与契合**

处理宝物最基础的规则问题，例如：

- 品级
- 品质
- 契合
- 认主
- 温养
- 使用门槛

### `artifact-treasures.md`
**器物型宝物**

处理已经成型、可被持有、祭炼、认主和传承的器物类宝物。

### `natural-treasures-and-oddities.md`
**天然宝物与异宝**

处理那些并非人为炼成、却本身就具有独立价值的天然宝物、奇物与异宝。

### `natal-treasures.md`
**本命宝物**

处理与修士个体深度绑定、可长期温养并与道路相关联的本命宝物。

### `strategic-treasures.md`
**战略重宝**

处理与组织、宗门、王朝或一方势力绑定的重宝与底蕴之物。

### `treasure-codex/`
**宝物条目库**

用于后续收录具体宝物条目。  
当前阶段主要保留内部规则与模板接口。

建议按以下顺序阅读：

1. `treasure-system.md`
2. `grades-and-compatibility.md`
3. `artifact-treasures.md`
4. `natural-treasures-and-oddities.md`

## 使用方式

阅读本目录时，建议先分清：

1. 宝物体系与百艺体系的边界  
2. 不同类型宝物各自处理什么  
3. 哪些内容应进入具体宝物条目，而不是继续停留在总纲层
