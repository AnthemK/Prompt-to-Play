# Status, Legitimacy, and Recognition

本文档处理本项目中不同类型势力的**地位、正当性与承认关系**。

这里的重点不是现代法理意义上的合法性，而是更符合仙侠世界的问题：

- 什么样的势力会被视为正统、名门、附庸、僭越或异类
- 一个势力为什么能“站得住”
- 名义上的承认，与现实中的畏服、利用、默认，有何不同
- 为什么有些势力很强，却始终不得人心；有些势力已衰，却仍有余威

本文档只讨论一般规则，不讨论具体势力个案。

## 本文档负责什么

本文档主要回答：

- 一个势力的地位通常由哪些因素决定
- “正统”“名门”“旁门”“附庸”“僭越”在本项目中的大致含义是什么
- 承认通常如何形成，又如何崩塌
- 名分与实力之间通常如何互动

## 本文档不负责什么

本文档不展开：

- 各类势力的类型划分  
  - 见 `order-types.md`
- 各类势力能管什么、管到哪  
  - 见 `authority-and-jurisdiction.md`
- 各类势力依赖什么资源关系  
  - 见 `dependency-and-resource-relations.md`
- 各类势力如何冲突、结盟与互相干预  
  - 见 `conflict-alliance-and-intervention.md`
- 某个具体势力的名望、声誉与历史地位  
  - 见对应势力档案

## 一、基本区分

在本项目中，至少区分以下几种状态：

### 1. 地位
指一个势力在天下秩序中的大致层级与分量。

### 2. 正当性
指它为什么被认为“有资格存在、有资格行事、有资格占位”。

### 3. 承认
指其他势力、地方共同体或社会舆论是否默认它的存在、边界与说法。

### 4. 声望
指它在更广泛范围内积累的名气、威慑与口碑。

这四者相关，但并不相同。

## 二、常见地位来源

一个势力的地位，通常来自以下几项中的若干组合：

- 传承与法统
- 历史与旧例
- 实力与战绩
- 辖地与资源
- 门风与名声
- 盟友与承认
- 对整体秩序的实际作用

其中，单有实力，不足以形成稳定地位；  
单有名义，也不足以长期维持承认。

## 三、常见类别

### 1. 正统
指被广泛视为“有资格代表某种正面秩序”的势力。

正统不等于完美，也不等于永远正确。  
它更多意味着：

- 传承可追溯
- 名分较稳
- 被较大范围默认承认
- 行事有一套可被理解的边界

### 2. 名门
指在实力、传承、门风或历史地位上都较为稳固，因而拥有较高社会评价的势力。

名门通常比一般正统势力更重“体面”与“声誉”。

### 3. 旁门
指存在有其合理性，但不处于主流秩序核心地带的势力。

其常见特征是：

- 传承不入主流
- 行事方式不同于大宗正统
- 可能强，也可能弱
- 常被使用、拉拢、提防，而不被完全接纳

### 4. 附庸
指自身不足以独立站稳，必须依附于更高位势力才有稳定地位的组织或共同体。

### 5. 僭越
指其行为、称号、边界或姿态，被他方视为超出了其应有位置。

“僭越”往往不是力量问题，而是**位置问题**。

## 四、承认通常如何形成

一个势力通常在以下条件下较容易获得承认：

### 1. 有可追溯的来路
例如传承、旧例、册封、长期占位、明确归属。

### 2. 有稳定的现实能力
它不仅能出现，还能持续维持秩序、保护地盘、承担后果。

### 3. 有可理解的边界
外界知道它大致管什么、不管什么，何时出手，何时收手。

### 4. 有被承认的价值
例如守边、育才、维稳、通商、镇压、调停、供给关键资源。

## 五、承认通常如何崩塌

一个势力通常会因为以下原因失去承认：

### 1. 名分与现实长期脱节
名义很高，现实却已无力支撑。

### 2. 行为不断越位
频繁越界、僭称、失礼、破坏默认边界。

### 3. 不能承担自己的位置
占着高位，却不能保护附庸、维持秩序、兑现承诺。

### 4. 被更高位秩序公开否定
一旦失去关键背书，其地位可能迅速塌陷。

## 六、名分与实力的关系

在本项目中：

- 没有实力，名分很难长久
- 只有实力，没有名分，也难以稳定扩张

实力决定你能不能站住。  
名分决定别人愿不愿意默认你站在那里。  
两者兼具，地位才稳。

## 七、判断一个势力“站不站得住”的简要标准

判断一个势力是否真正站得住，可优先看四点：

1. 它的来路是否可被解释  
2. 它的地位是否被周边默认承认  
3. 它是否能承担与其位置相称的责任  
4. 它的名分是否有现实力量支撑  

四者兼具，才算真正稳固。  
只靠祖上余威，不算稳。  
只靠一时强势，也不算稳。

## 八、最后说明

本文档的目的，是为后续势力档案提供一个前提：

- 地位不是自称得来
- 正统不是口号
- 承认不是虚礼
- 名分与现实，必须彼此咬合

后续具体建档时，应优先判断一个势力：

- 它凭什么被承认
- 谁承认它，谁只是怕它
- 它的地位是稳固的，还是悬空的
- 它最怕失去的是实力、名分，还是默认承认
