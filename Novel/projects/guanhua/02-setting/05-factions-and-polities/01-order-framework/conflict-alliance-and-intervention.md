# Conflict, Alliance, and Intervention

本文档处理本项目中不同类型势力之间的**互动方式**。

这里的重点不是具体历史事件，而是一般性的互动框架：

- 势力之间通常为何发生冲突
- 势力之间通常如何结盟
- 上位势力通常在什么情况下介入下层事务
- 什么样的摩擦会被局部化，什么样的摩擦容易升级

本文档只讨论一般规则，不讨论具体势力个案。

## 本文档负责什么

本文档主要回答：

- 不同类型势力之间常见的冲突来源是什么
- 不同类型势力之间常见的合作与结盟方式是什么
- 何种情况下会出现调停、庇护、借道、代理冲突、联合镇压
- 上位势力介入地方事务时，通常依据什么
- 一个局部事件通常如何扩大，或如何被压住

## 本文档不负责什么

本文档不展开：

- 各类势力的类型划分  
  - 见 `order-types.md`
- 各类势力能管什么、管到哪  
  - 见 `authority-and-jurisdiction.md`
- 各类势力依赖什么资源关系  
  - 见 `dependency-and-resource-relations.md`
- 各类势力为何被承认、视为正统或视为僭越  
  - 见 `status-legitimacy-and-recognition.md`
- 某场具体战争、盟约、背叛或事件经过  
  - 见对应历史文件或势力档案

## 一、基本判断

在本项目中，势力互动通常不只由“善恶”决定，而主要由以下因素推动：

- 利害
- 名分
- 边界
- 资源
- 旧怨
- 现实成本
- 上位秩序是否允许

因此，冲突与联盟都应被视为**结构性结果**，而不只是人物情绪的外化。

## 二、常见冲突来源

### 1. 资源冲突
最常见，也最稳定。

例如：

- 矿脉
- 药山
- 坊市
- 渡口
- 商路
- 秘境入口
- 人才来源

资源冲突往往最容易持续，也最容易披上名分外衣。

### 2. 边界冲突
不同势力的势力范围、影响范围、驻点与默认地盘发生重叠。

这类冲突常表现为：

- 越界
- 擅入
- 私设据点
- 强行收人
- 插手他方属地事务

### 3. 名分冲突
表面资源未必立刻冲突，但一方的行为被另一方视为：

- 僭越
- 失礼
- 夺统
- 越位
- 破坏旧例

这类冲突在仙侠世界里尤其重要，因为很多秩序依赖“默认承认”。

### 4. 依附冲突
某个原本附属于一方的地方势力、豪族、坊市、边地节点，转而投向另一方。

这类冲突表面看是“叛附”，本质上是秩序网络的重新编排。

### 5. 代理冲突
高位势力不直接出手，而通过地方势力、附庸、边军、商盟、客卿或代言人推进争夺。

这类冲突最容易长期化，也最容易表面降温、实则升级。

## 三、常见联盟形式

### 1. 利益联盟
以资源、通道、互市、互保为核心。

这类联盟最常见，也最脆弱。  
利益还在时稳定，利益逆转时最先破裂。

### 2. 对敌联盟
面对更强外部压力时，原本互相提防的势力暂时合作。

这类联盟通常目标明确、期限有限，很少真正消除旧怨。

### 3. 名分联盟
通过盟约、册封、联姻、共推某种秩序口号来获得更高正当性。

这类联盟不一定亲密，但能降低彼此行动成本。

### 4. 依附联盟
弱势一方向强势一方靠拢，以换取：

- 庇护
- 资格
- 市场
- 资源准入
- 对外背书

这类联盟往往带有明显上下关系。

## 四、常见介入方式

### 1. 调停
上位势力不直接吞并，而是压下争端、划线、追认既成事实。

适用于成本高、但事态尚可收束的局面。

### 2. 背书
某方公开支持某地方势力、某继承人、某组织安排，使局势迅速倾斜。

### 3. 借道与驻点
以临时协防、交易保障、秘境看守、边防需要等理由进入他方地带。

短期是介入，长期则可能转化为事实控制。

### 4. 联合镇压
面对越界者、失控者、边患或会威胁整体秩序的对象，多方暂时联手。

### 5. 代理施压
不直接出手，但切断资源、封锁通道、抬高成本、扶植对手，使目标自行失衡。

## 五、局部事件如何升级

一个局部事件通常会因为以下原因升级：

### 1. 触碰关键节点
若事件发生在矿脉、坊市、边关、秘境入口等关键位置，升级概率会明显提高。

### 2. 牵涉名分
若某一方认为退让会损害其地位、脸面、正统性或默认边界，局势更难缓和。

### 3. 有外部接应
一旦地方冲突背后站着更高位势力，事件就不再只是地方事务。

### 4. 成本被错误估计
一方误判另一方底线、误判上位势力是否会介入，常导致失控升级。

## 六、局部事件如何被压住

一个局部事件通常会因为以下原因被压住：

### 1. 各方都不想扩大战线
收益有限，成本过高。

### 2. 上位势力明确划线
一旦更高位秩序表态，地方势力往往会退回可接受范围。

### 3. 可通过补偿解决
若冲突本质仍是利益冲突，而非名分冲突，赔偿、让渡、换约通常有效。

### 4. 任何一方都无法速胜
当速胜无望时，最理性的选择往往是收手而不是硬打。

## 七、判断一场冲突“会不会扩大”的简要标准

判断一场冲突是否会升级，可优先看四点：

1. 它是否触碰关键资源或关键边界  
2. 它是否伤及某方名分与默认地位  
3. 是否已有更高位势力卷入  
4. 扩大战线后，是否有人能真正获利  

四者越多，越容易升级。  
若四者皆弱，则更可能停留在局部摩擦。

## 八、最后说明

本文档的目的，是为后续势力档案提供一个前提：

- 冲突不是偶然的
- 联盟不是永恒的
- 介入不是中性的
- 很多局势变化，首先不是看谁更强，而是看谁更有理由、更有位置、更有成本优势去出手

后续具体建档时，应优先判断一个势力：

- 最常和谁冲突
- 最可能和谁合作
- 最怕谁介入
- 最常以什么方式介入别人
