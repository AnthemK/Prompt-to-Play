# 05 Factions and Polities

这个目录处理本项目中“势力—政体—现实秩序载体”的部分。

本层的核心单位不是抽象制度，而是**具体势力**。  
也就是说，很多设定不是先有统一制度，再落到世界上；而是先有某个宗门、世家、王朝、学宫、商盟、边军体系或异族政体，然后才进一步生长出它的辖地、规则、资源、社会状态、关键人物与剧情功能。

因此，本目录采用：

- **以具体势力/政体为中心建档**
- **以总图与索引文件补充共性规律**

## 本目录负责什么

本目录主要回答：

- 这个世界有哪些主要势力与政体
- 它们分别属于什么类型
- 它们如何承载秩序、占有资源、塑造社会
- 主角与其他角色会接触哪些层级的势力

## 本目录不负责什么

以下内容不在本目录展开：

- 世界底层规则与总约束  
  - 见 `01-constitution/`
- 天地、道则、灵机、界域、飞升、天劫等本体问题  
  - 见 `02-cosmos-and-principles/`
- 修炼体系、功法、法术、法宝、丹符阵器等修行技术问题  
  - 见 `03-cultivation-and-arts/`
- 地理格局、地图、历史时间线与时代分期  
  - 见 `04-world-layout-and-history/`
- 通用术语、命名规范、风格表与未决问题  
  - 见 `06-reference/`

## 组织方式

本目录分为两部分：

- **总图与索引**
  - 用于总结主要秩序类型与共性规律
- **具体势力档案**
  - 用于分别建立宗门、世家、王朝、学宫、商盟、边军体系、异族政体与地方势力的档案

本 README 只展开到下一层。  
下一层的具体内容、文件结构与阅读顺序，请查看对应子目录中的 README。
