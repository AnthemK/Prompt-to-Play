## 卷一意义

这里是第一卷最常用的设定层。

它直接决定：

- 主角平时怎么活
- 主角如何被筛选
- 主角如何被看见
- 主角如何第一次接触宗门真实面貌
- 主角如何在整卷中一步步走到大比前列
