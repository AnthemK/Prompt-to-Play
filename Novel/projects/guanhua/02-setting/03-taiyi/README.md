# 03 Taiyi

这个目录是太一门设定专章。

## 这个目录解决什么问题

它主要回答：

- 太一门是什么样的宗门
- 它内部如何运转
- 外门弟子的生活和筛选机制是什么
- 太一门法脉、招法与大比、下山等事务如何成立

## 当前分层

- `01-overview/`
  - 宗门总览与命名比较
- `02-organization/`
  - 峰脉、殿阁、运行结构、组织边界
- `03-curriculum-and-operations/`
  - 外门生态、课程、制度事件、日常运转
- `04-lineages/`
  - 太一门法脉与招法谱系
- `assets/`
  - 太一门印象图与示意图

## 给协作者的阅读方法

先读本目录 README。

如果你只做卷一相关实验，这里是最重要的设定入口。

如果你要继续深入，请按问题进入下一层：

   - 外门生态、课程、功点、下山、大比、近景手感
   - 进入后先读 `03-curriculum-and-operations/README.md`
4. `04-lineages/`
   - 太一门法脉、招法与法门偏好
   - 进入后先读 `04-lineages/README.md`
