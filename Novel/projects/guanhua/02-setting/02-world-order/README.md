# 02 World Order

这个目录放太一门之外、但会压到主线的天下结构。

## 这个目录解决什么问题

它主要回答：

- 天下有哪些稳定势力类型
- 太一门之外的秩序背景是什么
- 不同种族、文明与阵营冲突怎样成立

## 文件索引

- `power-structures.md`
  - 势力类型总图
- `factions-and-orders.md`
  - 阵营、秩序与主要对手类型
- `notable-sects.md`
  - 太一门之外的重要宗门样本
- `species-and-civilizations.md`
  - 种族与文明结构

## 给协作者的阅读建议

先把这里当作卷一背景板，而不是当前主战场。

卷一以太一门内部为主，这里的内容主要负责：

- 保证世界不是空壳
- 为后续扩展留真实接口
- 约束太一门在天下中的位置
