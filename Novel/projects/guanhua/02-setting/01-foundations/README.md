# 01 Foundations

这个目录放《观化》的世界底层规则。

## 这个目录解决什么问题

它主要回答：

- 世界底层规则是什么
- 修炼体系与法宝体系怎样成立
- 命名风格有什么统一要求
- 哪些规则一旦进入正文就必须稳定追责

## 文件索引

- `world-core.md`
  - 世界观底层骨架
- `cultivation-system.md`
  - 经典八境与修炼逻辑
- `arts-and-artifacts.md`
  - 功法、招数法门、法宝等技术体系
- `naming-and-samples.md`
  - 命名样本库
- `history-and-timeline.md`
  - 会压到主线的历史断点
- `rule-ledger.md`
  - 规则账本
