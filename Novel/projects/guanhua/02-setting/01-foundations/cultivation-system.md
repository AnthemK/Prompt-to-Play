# Cultivation System

## 用途

这份文档只定义《观化》的修炼体系框架。

它回答四个问题：

1. 修炼的本质是什么。
2. 境界为何会塑造秩序。
3. 经典八境在本书中分别意味着什么。
4. 后续功法、法宝、斗法和势力设定应服从哪些修炼层面的总规则。

## 一、修炼的本质

《观化》的修炼，不是单纯吸纳灵气、堆高数值。

它同时是三件事：

1. 承载能力的提升  
人逐步获得承受更高层级天地之力的资格。

2. 资源条件的筛选  
修炼永远依赖功法、环境、护持、时间与组织接纳。

3. 认知层级的上升  
境界越高，通常看得越远、知道得越多、承担的判断后果也越重；但这并不自动消除偏见、执念、立场与路径依赖。

因此，修炼既是超凡化过程，也是秩序分层过程。

## 二、境界为何重要

在《观化》里，境界不是简单战力标签，而是四种变化同时发生：

1. 身心结构改变  
身体、神识、法力与天地的关系发生重构。

2. 行动能力改变  
可使用的术法、法宝、调度范围与生存方式发生变化。

3. 资源门槛改变  
越高境界，越依赖更稀缺、更匹配、更不可标准化的条件。

4. 社会位置改变  
信息权限、组织责任与行为后果同时上升。

所以大境界突破，既是修炼飞跃，也是社会飞跃。

## 三、经典八境

《观化》采用经典八境：

- 练气
- 筑基
- 金丹
- 元婴
- 化神
- 合体
- 渡劫
- 大乘

使用经典框架，不是为了偷懒，而是为了把重点放在：

- 每一境的真实含义
- 每一境对应的资源门槛
- 每一境在秩序中的位置

## 四、八境的抽象定义

### 1. 练气

修行之始。

标志是能够感应并引动灵机，让人从纯凡俗状态进入修行秩序。

这是人数最多、淘汰最多的一层，也是凡人与修士的第一道分界。

能力特征：

- 能引气入体，初步洗练身心
- 能使用基础术法、基础法器和最低层级的修行工具
- 神识至多处于萌芽或极弱状态，难以稳定外放
- 已明显强于凡俗，但仍高度依赖环境、功法与他人护持

边界：

- 还不能稳定脱离组织与资源链独立生存
- 通常不能辟谷，也不能长期御器远行
- 对高强度斗法、远行和长期闭关的承受力都有限
- 所见世界仍以局部生存与眼前规矩为主

### 2. 筑基

根基成形。

标志是修炼结构开始稳定，修士真正具备长期培养价值。

这是大多数组织区分“外围”与“正式培养对象”的第一道硬门槛。

能力特征：

- 气机运转趋于稳定，修行不再全靠外力扶持
- 神识真正成形，可以支撑探查、御器与较复杂的术法调度
- 可以较稳定地驱使成体系的术法与器物
- 已具备承担基层治理、守备、执事和长期任务的能力

边界：

- 仍属于体系中的正式骨干，而非真正中坚
- 通常已可辟谷，并具备中短程持续行动能力，但还谈不上真正自由远游
- 资源与路数差异会在这一层开始快速放大
- 对更高层规则的理解仍主要来自服从，而非参与解释

### 3. 金丹

个人修行核心凝成。

标志是修士不再只是体系中的基础构件，而开始成为可独当一面的中坚力量。

从这一层开始，个人道途、法宝体系和上限差异会被明显放大。

能力特征：

- 个人法力核心稳定，斗法风格开始成形
- 已能较稳定地进行长距离遁行，并承载本命法宝体系
- 可较长期支撑护身、攻伐、遁行与御器体系
- 已能在一地、一脉或一类事务中独当一面

边界：

- 仍深受组织资源、法宝质量与路数高低制约
- 可以主事，但通常还不足以改写大范围秩序
- 丹成之后，上限与路数开始真正固化，许多错误会变得难以回头
- 个人上限开始清晰，道途分化也真正开始不可逆

### 4. 元婴

生命与神识层面的深度跃迁。

标志是个人伟力开始稳定压过常规组织尺度，并足以对局部秩序产生实质影响。

这一层通常也是高层政治开始真正关心你的起点。

能力特征：

- 神识、法力、生命层次出现明显质变
- 元婴与肉身、神魂之间已可形成更复杂的分离与保命结构
- 可在较大范围内稳定压制低阶修士数量优势
- 已能成为一方高层、重镇支柱或重大博弈中的关键变量

边界：

- 虽能压过常规组织尺度，但仍无法脱离更大秩序与更高层威慑
- 这一层以后，跨大境界取胜应极罕见，只能依赖极强准备或极端克制
- 这一层最容易把局部经验误当全局真理
- 个人意志开始外溢为政治后果，但尚不足以定义天下方向

### 5. 化神

对天地、人间与自身关系的统合进一步加深。

标志是不只是更强，而是开始成为秩序解释者、方向制定者和大局变量。

从这一层起，认知、立场和路径依赖的重要性会继续上升。

能力特征：

- 对天地之势、法门本质和大局结构的把握显著提升
- 开始能以自身神意统摄局部天地之势，形成更明确的法域、压制与规则偏向
- 行动已不只是个人出手，而会牵动一方秩序、一路法统或一片疆域
- 开始真正拥有解释规则、制定方向和塑造局面的能力

边界：

- 越往上越不是单靠资源就能堆出来
- 这一层的偏见、执念与选择，后果会被成倍放大
- 虽能参与天下格局，但仍不是不可替代的终局力量

### 6. 合体

身、神、法与所修之路高度统一。

标志是修士本身已成为某种战略性存在，其寿命、威慑与存在方式都会外溢为组织资源。

这是典型的顶级势力支柱层，因为其寿命、威慑、稳定性、护持能力都外溢为组织资源。

能力特征：

- 人与所修之路高度契合，力量、寿命与威慑都趋于战略化
- 许多此前分离的化身、外化手段或道路分支，会在这一层被重新统合
- 其存在本身即可稳定一方秩序、压住一脉气运或改写高层博弈方式
- 已是顶级组织的重要支柱和长期筹码

边界：

- 数量极少，且高度依赖组织与道途的长期匹配
- 其代价不再只是个人风险，而常常变成整条线的沉没成本
- 仍处于天地可容纳的高位之内，并未抵达最终关口

### 7. 渡劫

对既有道途的极限检验。

它不是稳定常备层，而是高风险的关口层。

这一阶段的成败，往往不只决定个人命运，也会牵动势力结构与时代平衡。

能力特征：

- 已逼近当前天地秩序所能容纳的上限
- 一次进退，都会对法统、组织和高层均势造成连锁影响
- 其护道、布置、时机与因果条件本身，就是天下级资源调度

边界：

- 这不是可量产的稳定层级，而是高度危险的过关状态
- 成败都可能留下远超个人的后果
- 任何轻率越关，都会把代价推到极高

### 8. 大乘

当前天地秩序内的极高完成度。

这类存在本身就会反过来塑造资源流向、组织边界和时代气候。

数量极少，其存在本身就是天下大事。

能力特征：

- 已是时代尺度上的顶层存在
- 法力、肉身、道路与天地容纳度都已逼近飞升前的极限完成
- 其出手、沉默、站位和存亡，都会改变天下预期与长期格局
- 本身已接近秩序的一部分，而不只是秩序中的强者

边界：

- 数量必须极少，且不能稳定量产
- 再往上的问题，已不再属于常规修行社会的日常结构
- 这类存在更像时代边界，而不是常规干部层级

## 五、八境的社会分布

八境在社会中的大致位置，应保持如下结构：

- 练气：基础人口层
- 筑基：正式骨干层
- 金丹：中坚与地方强者层
- 元婴：高层与局部秩序改写层
- 化神：天下格局参与层
- 合体以上：时代支柱与顶层变量层

这不是僵硬官阶表，而是世界尺度的稳定参照。

## 六、修炼体系必须遵守的总规则

### 1. 低阶人数多，高阶人数少

修仙世界可以缓慢增长，但人口结构必须总体稳定。

高阶修士不能多到冲垮世界，也不能少到无法支撑长期秩序。

### 2. 同境差距必须可解释

同一境界内部，差距可以很大，但通常不应轻易跨越社会层级所依托的资源、法宝、传承和组织支持差距。

但差距必须主要来自：

- 功法
- 法宝
- 积累
- 经验
- 准备
- 克制
- 意志与林场判断

不能来自作者任意偏置。

### 3. 跨境差距必须真实存在

同境内部是连续变化，跨越大境界则应形成真实飞跃。

少数特殊情况可以缩小差距，但不能抹平基本断层。

### 4. 越往高境，越依赖匹配性

高境突破越来越不是“资源够不够”的问题，而是“资源、功法、环境、心性与时机是否匹配”的问题。

因此，越高境界越难标准化、越难量产。

### 5. 境界提升通常伴随信息权限上升

修士不是境界一高就自动正确，但通常会接触到更高层级的知识、责任和视角。

因此，修炼体系本身也是世界信息逐层展开的结构基础。

### 6. 一切破境都必须带有代价与风险

破境不是机械升级，而是一次重新分配代价、资源和未来路径的过程。

越往高境，失败的代价越大，成功的后果也越广。
