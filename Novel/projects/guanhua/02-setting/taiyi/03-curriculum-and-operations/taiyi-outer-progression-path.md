# Taiyi Outer Progression Path

## 用途

定义卷一可直接调用的外门成长路径。

这份文档不替主角写剧情，只回答：

- 陆守一这类外门低阶弟子，怎样成长才合理
- 卷一中的修炼进步主要靠什么获得
- 为什么他能在卷末具备夺魁资格，而不破坏稀缺性与真实感

## 一、核心判断

卷一若以“外门无名弟子最终在大比夺魁”为结局，就必须避免两种失真：

### 1. 只靠苦修直线升级

这样会把宗门制度、任务履历和师长提携全部写虚。

### 2. 只靠奇遇突然暴起

这样会破坏《观化》最重要的真实感。

因此更合理的成长路径应由四部分叠加：

1. 基础根基确实比同层更稳
2. 有限而关键的资源补给
3. 在任务与见识中获得判断力提升
4. 师长给出真正契合的法门，而不是直接代打

## 二、卷一起点与终点的大致要求

### 起点

陆守一当前稳定位置是：

- 十四岁
- 练气三层
- 只有基础法门
- 没有真正门内底蕴加持

### 卷一末应达到的状态

更合理的卷一末状态，不应是夸张跃迁，而应是：

- 修为进入练气中后段
- 基础根基远比账面修为更扎实
- 具备一套可以打完整场的攻守体系
- 有过真实任务履历
- 被多个不同系统的人重新估价

换句话说，卷一末的他应是：

- 名次显赫
- 但仍然没有脱离外门逻辑

这样第二卷才有继续向上爬的空间。

## 三、卷一成长必须经过的五道台阶

### 第一阶：把低阶基础练到能活

这一阶最重要的，不是涨修为，而是：

- `太一引气诀` 更稳
- `镇息诀` 真正练到收得住
- `守中身法` 从会用变成能在慌乱时用
- `定元护身印` 不再只是勉强成形

这一层的现实意义是：

- 让他不容易在杂务、切磋和小摩擦里被打散

### 第二阶：争到第一批真正有用的资源

卷一前中段，主角最该争的不是大机缘，而是：

- 一段较完整的修炼时段
- 一两次补额丹药
- 一次更干净的抄录机会
- 一次不被苦差拖垮的轮值安排

这类资源看似不惊人，却决定了：

- 他能不能从练气三层稳定抬到四层乃至五层

### 第三阶：获得第一门真正改变打法的法门

周既白若要传法，最合理的顺序仍是：

- 先传 `澄心观息法`

原因不是它最强，而是它最适合陆守一。

它应给主角带来的，不是爆发，而是：

- 看得更清
- 气机更稳
- 更善于在乱局里抓真正关键处

这会把他的优势从“隐约有”变成“可被放大”。

### 第四阶：在任务中把“稳”转化为“能赢”

仅有静态修炼仍不够。

卷一必须让主角在见习任务与下山经历中获得：

- 对距离、节奏、风险和人心的判断
- 对宗门法令与现实后果之间落差的理解
- 对自己底线的第一次较重检验

这一步之后，他才不只是“基础扎实”，而是：

- 真正具备比赛和办事两用的能力

### 第五阶：补上第一门明确攻伐或制敌法门

更适合作为卷一后段发力点的，不是过于花哨的新招，而是：

- `镇虚手`

它若写得好，价值不在华丽，而在：

- 起手短
- 制人气机
- 适合以稳破急
- 与 `澄心观息法` 形成内外呼应

这会让陆守一形成一套很符合人物气质的卷一战斗结构：

- 先稳
- 再看
- 再准
- 最后以短而狠的变化定局

## 四、卷一成长的资源来源

### 1. 宗门公开资源

包括：

- 月例
- 功点换补额
- 基础抄录额度
- 更好的修炼时段

这部分只能让他活下来和慢慢涨。

### 2. 事务表现带来的隐性资源

包括：

- 被记住
- 被少塞苦差
- 被认为可靠
- 在丙类任务摸底时不会被直接排除

这类资源很难量化，却比单纯丹药更重要。

### 3. 师长识人后的定向投入

包括：

- 一门合适法门
- 一次纠偏
- 一句足以改名册印象的判断

要强调：

- 这不是开后门
- 而是宗门对稀缺苗子的有限再投入

### 4. 危险见识带来的认知收益

下山见习后，主角应获得的最珍贵东西并不一定写成宝物。

更合理的是：

- 对局势的判断更实
- 对人心与立场的理解更深
- 因此斗法与办事都更少犯错

## 五、为什么大多数外门弟子走不到这里

为了维持稀缺性，必须明确：

### 1. 很多人第一阶都没练透

基础法门会用，不等于遇事不乱。

### 2. 很多人第二阶就被时间与轮值磨掉

有些人并不差，只是没有完整修炼时段。

### 3. 很多人拿不到第三阶的契合法门

不是每个人都能碰到既识人又肯承担的师长。

### 4. 很多人第四阶会在真实任务里露底

能修不等于能办事。

### 5. 很多人第五阶前就被格局压住

大比前，外门的热门、派系和信息差本身就是门槛。

## 六、卷一末“夺魁”为什么仍然合理

只要满足以下条件，陆守一在卷一末夺魁就是合理的：

### 1. 他不是账面修为最高的人

而是：

- 根基更稳
- 路子更整
- 临场判断更成熟

### 2. 大比本来就不只看纯修为高低

它还看：

- 根基
- 可用性
- 心性
- 攻守完整度

### 3. 他拿到的是“非常合身”的成长，不是“超规格”的成长

这点非常重要。

陆守一卷一最该得到的，不是别人拿不到的神物，而是：

- 少数人看得懂
- 真正适合他
- 因而能把他的内秀优势放大的那一类东西
