# Projects

这个目录放正在推进的小说项目。

## 规则

- 一个项目一个目录
- 目录名用稳定的英文或拼音 slug
- 中文书名、别名、题材标签写在项目内 `README.md`
- 新项目优先从 `../shared/templates/project-template/` 复制

## 示例

```text
projects/
  immortal-bureau/
  apocalypse-farm/
  court-of-embers/
```
